# 运行与环境配置指南

**环境**：Python 3.11+（推荐 3.12）；使用二期 Web 控制台时需 **Node.js 18+**。  
**约定**：后端默认 **`127.0.0.1:8076`**（与 `web` 开发代理一致）。工作目录均为**项目根目录**（含 `app/`、`requirements.txt` 的那一层）。

---

## 第一次运行

在项目根目录打开 **PowerShell**，依次执行。

### 仅后端 API

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
copy .env.example .env
```

用编辑器打开 `.env`，至少填写 **`LLM_API_KEY`**；非 OpenAI 官方时按需改 **`LLM_BASE_URL`**、**`LLM_MODEL`**。其它开关见 `.env.example` 注释。

```powershell
python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8076
```

- Swagger：`http://127.0.0.1:8076/docs`  
- 探活：`http://127.0.0.1:8076/health`

### 后端 + 二期 Web 控制台

完成上一节「仅后端」到 **启动 uvicorn 之前**（含 `copy .env`、编辑 `.env`），再**另开一个终端**执行：

```powershell
cd web
npm install
npm run dev
```

浏览器一般打开 `http://127.0.0.1:5173`。若后端不是 8076，先在 `web` 目录复制 `web/.env.example` 为 `web/.env`，设置 `VITE_DEV_API_TARGET=http://127.0.0.1:<你的端口>` 后再 `npm run dev`。

## 后续运行

### 仅后端

```powershell
.\.venv\Scripts\Activate.ps1
python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8076
```

（修改 `.env` 后需**重启**该进程。）

### 后端 + Web

终端 1（项目根）：

```powershell
.\.venv\Scripts\Activate.ps1
python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8076
```

终端 2：

```powershell
cd web
npm run dev
```
