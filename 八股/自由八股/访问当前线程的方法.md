你问的「访问当前线程」一般就是指 **在 Java 程序里如何拿到正在执行的线程对象，以及如何获取当前线程的相关信息**。

------

# 🔹 1. 获取当前线程对象

```java
Thread t = Thread.currentThread();
```

这是最常用的方法，返回正在执行的线程引用。

示例：

```java
System.out.println(Thread.currentThread().getName()); 
```

输出类似：

```
main
```

表示当前运行在 `main` 主线程。

------

# 🔹 2. 获取当前线程信息

通过 `Thread.currentThread()` 可以访问许多属性/方法：

- `getName()`：线程名
- `getId()`：线程 ID
- `getPriority()`：优先级
- `getState()`：线程状态（NEW、RUNNABLE、BLOCKED、WAITING、TIMED_WAITING、TERMINATED）
- `isAlive()`：是否还存活
- `isDaemon()`：是否是守护线程
- `isInterrupted()`：是否被中断

示例：

```java
Thread t = Thread.currentThread();
System.out.println("Name: " + t.getName());
System.out.println("Id: " + t.getId());
System.out.println("Priority: " + t.getPriority());
System.out.println("State: " + t.getState());
System.out.println("Is Daemon: " + t.isDaemon());
```

------

# 🔹 3. 当前线程的控制操作

对当前线程可以调用的一些方法：

- `Thread.sleep(ms)`：让当前线程休眠一段时间。
- `Thread.yield()`：当前线程让出 CPU 时间片，调度器可以切换到别的线程。
- `Thread.interrupted()`：检查并清除当前线程的中断标志。

示例：

```java
try {
    Thread.sleep(1000); // 当前线程休眠 1 秒
} catch (InterruptedException e) {
    System.out.println("Current thread interrupted");
}
```

------

# 🔹 4. ThreadLocal（线程本地变量）

如果你想**给当前线程存储和获取独立变量**（其他线程互不影响），可以使用 `ThreadLocal`：

```java
private static ThreadLocal<String> threadLocal = new ThreadLocal<>();

public static void main(String[] args) {
    threadLocal.set("value for " + Thread.currentThread().getName());
    System.out.println(threadLocal.get()); 
}
```

这也是一种“访问当前线程上下文”的方式。

------

✅ **总结**
 在 Java 中访问当前线程主要用 **`Thread.currentThread()`**。
 通过它可以获取：

- 当前线程对象本身
- 名称、ID、状态、优先级等属性
- 控制当前线程（sleep、yield、interrupted）
   如果需要存储每个线程自己的数据，可以用 **`ThreadLocal`**。
