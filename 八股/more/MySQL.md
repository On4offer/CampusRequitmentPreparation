------

## 一、MySQL 是什么？

**MySQL** 是一个开源、免费的、**关系型数据库管理系统（RDBMS）**，使用**结构化查询语言（SQL）**进行数据操作。
 它是全球最流行的数据库之一，广泛用于网站、应用、后台系统等。

------

## 二、MySQL 的核心特点

| 特点       | 说明                                             |
| ---------- | ------------------------------------------------ |
| 开源免费   | 社区版永久免费使用，企业版收费                   |
| 使用 SQL   | 标准关系型数据库语言，易学易用                   |
| 跨平台支持 | 可运行在 Windows、Linux、macOS 等多种平台上      |
| 性能优越   | 支持百万级数据的查询、事务操作                   |
| 支持事务   | 提供事务管理、回滚、提交、ACID 等（InnoDB 引擎） |
| 可扩展性强 | 可与 Java、PHP、Python、Node.js 等语言无缝集成   |

------

## 三、MySQL 中的核心概念

### 1. 数据库（Database）

- 一组结构化的数据集合，比如一个网站可以有一个数据库。

### 2. 表（Table）

- 数据的具体存储结构，类似 Excel 表格。

### 3. 行（Row）和列（Column）

- 行：一条记录
- 列：一个字段，如姓名、年龄等

### 4. 主键（Primary Key）

- 唯一标识一条记录的字段，如 id

### 5. 外键（Foreign Key）

- 表与表之间的关联关键字段

------

## 四、MySQL 的常见 SQL 操作

```sql
-- 创建数据库
CREATE DATABASE school;

-- 选择数据库
USE school;

-- 创建表
CREATE TABLE student (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(50),
    age INT
);

-- 插入数据
INSERT INTO student (name, age) VALUES ('Alice', 20);

-- 查询数据
SELECT * FROM student;

-- 更新数据
UPDATE student SET age = 21 WHERE name = 'Alice';

-- 删除数据
DELETE FROM student WHERE id = 1;
```

## 五、MySQL 的存储引擎

| 引擎   | 特点                                   | 是否支持事务 |
| ------ | -------------------------------------- | ------------ |
| InnoDB | 默认引擎，支持事务、外键、安全可靠     | 是           |
| MyISAM | 查询速度快，适合读取密集型应用         | 否           |
| MEMORY | 数据存储在内存中，极快，但数据不持久化 | 否           |

> 推荐：生产环境基本都用 **InnoDB**。

------

## 六、MySQL 的架构简图（简要理解）

```
[客户端] → [连接层] → [查询解析器/优化器] → [存储引擎] → [磁盘数据]
```

- SQL 会被解析优化，然后调用存储引擎（如 InnoDB）进行数据处理。
