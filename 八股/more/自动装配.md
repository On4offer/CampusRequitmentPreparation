### **自动装配（Autowiring）简介**

**背景**：在 Spring 框架中，**自动装配**是指由 Spring 容器**自动将依赖对象注入到需要它的组件中**，而无需手动配置。它是实现 **控制反转（IoC）** 和 **依赖注入（DI）** 的重要方式之一，极大地简化了开发工作。

------

### **一、自动装配的方式（主流 4 种）**

Spring 提供了多种自动装配方式，主要包括以下几种：

#### 1. **基于注解的自动装配**

##### `@Autowired`（最常用）

- 默认按照**类型**（byType）注入。
- 可以搭配 `@Qualifier` 指定具体的 Bean 名称。
- 作用于：构造方法、字段、Setter 方法。

```java
@Component
public class UserService {
    @Autowired
    private UserRepository userRepository;
}
```

##### `@Qualifier`

- 与 `@Autowired` 配合使用，**按名称注入**，用于解决多个同类型 Bean 的冲突。

```java
@Autowired
@Qualifier("mysqlUserRepository")
private UserRepository userRepository;
```

##### `@Inject`（来自 JSR-330，等价于 `@Autowired`，无 required 属性）

##### `@Resource`（来自 JDK，默认按**名称**注入）

```java
@Resource(name = "userRepository")
private UserRepository userRepository;
```

------

#### 2. **基于 XML 的自动装配（老方式）**

- 在 `<bean>` 配置中使用 `autowire` 属性：

```xml
<bean id="userService" class="com.example.UserService" autowire="byType" />
```

常见方式有：

- `no`（默认）不自动装配；
- `byName`：根据属性名匹配 Bean 名称；
- `byType`：根据类型匹配；
- `constructor`：通过构造器注入；
- `autodetect`（已废弃）：自动判断使用 byType 或 constructor。

------

#### 3. **构造方法注入（推荐）**

```java
@Component
public class OrderService {
    private final OrderRepository repo;

    @Autowired
    public OrderService(OrderRepository repo) {
        this.repo = repo;
    }
}
```

优点：

- 必须传入依赖（适合强依赖）；
- 更有利于测试；
- 更符合 “不可变性” 原则。

------

#### 4. **Setter 方法注入**

```java
@Component
public class ProductService {
    private ProductRepository productRepository;

    @Autowired
    public void setProductRepository(ProductRepository productRepository) {
        this.productRepository = productRepository;
    }
}
```

适合可选依赖、或后期可能变更的情况。

------

### **二、自动装配的原理简述**

Spring 在创建 Bean 时，会通过 **反射和类型推断** 找到需要注入的依赖，并：

1. 扫描注解（如 `@Component`, `@Autowired`）；
2. 找到匹配的 Bean（byType 或 byName）；
3. 如果找到多个 Bean，需要指定 `@Qualifier` 或抛出异常；
4. 最终通过反射将依赖注入到字段、构造器或方法中。

------

### **三、自动装配失败的处理**

默认情况下，如果 `@Autowired` 找不到对应 Bean，会抛出异常。

- 可通过 `required = false` 解决：

```java
@Autowired(required = false)
private EmailService emailService;
```

------

### **四、小结一句话**

> **自动装配是 Spring 框架中用于自动注入依赖对象的机制，常通过 `@Autowired` 实现，支持按类型、按名称等多种注入方式，极大地提升了开发效率和可维护性。**
