# Java开发实习生面试试卷（第一套）

## 一、基础题（每题5分，共40分）

### 1. Java基础与集合框架

1. 请简述ArrayList和LinkedList的底层数据结构及适用场景。（5分）

2. HashMap在JDK 1.7和JDK 1.8中的实现有哪些主要区别？为什么要引入红黑树？（5分）

3. 什么是哈希冲突？HashMap是如何解决哈希冲突的？（5分）

### 2. 多线程编程

4. synchronized和ReentrantLock的区别是什么？底层实现原理有何不同？（5分）

5. ThreadLocal的底层实现原理是什么？为什么key要用弱引用？（5分）

6. 线程池的核心参数有哪些？请简述线程池的工作原理。（5分）

### 3. SpringBoot基础

7. SpringBoot的自动装配原理是什么？@EnableAutoConfiguration注解是如何工作的？（5分）

8. Spring中的Bean生命周期包括哪些阶段？Spring是如何解决循环依赖问题的？（5分）

## 二、项目实战题（每题10分，共40分）

1. 请描述你在医疗美容行业管理系统中开发的Excel商品批量导入功能的实现流程。如何处理文件格式验证和错误处理？（10分）

2. 在小众点评项目中，你是如何使用Redis实现缓存层的？如何解决缓存穿透、缓存击穿和缓存雪崩问题？（10分）

3. 请解释你在项目中实现的审计日志功能。如何确保日志记录不影响主流程的性能？（10分）

4. 在高并发场景下，如何使用Redis实现分布式锁？会存在哪些潜在问题以及如何解决？（10分）
