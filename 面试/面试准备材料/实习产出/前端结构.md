
## 项目架构概览

基于 **Vue 2.5.9** + **Element UI 2.4.5** 的管理系统，采用模块化架构。

## 一、技术栈

### 核心框架
- Vue 2.5.9
- Vue Router 3.0.1
- Vuex 3.0.1
- Element UI 2.4.5

### 构建工具
- Webpack 3.8.1
- Babel
- ESLint

### 主要依赖
- axios：HTTP 请求
- echarts：图表
- fullcalendar：日历
- tinymce：富文本
- socket.io-client：WebSocket
- vue-i18n：国际化
- mockjs：Mock 数据

### 2. 路由架构 (`src/router/index.js`)

路由特点：
- 动态路由：基于权限动态生成
- 路由懒加载：开发环境直接导入，生产环境按需加载
- 多路由映射：支持不同角色（总部、库管、其他）的路由配置

路由类型：
- `constantRouterMap`：固定路由（登录、404、401等）
- `asyncRouterMap`：异步路由（需要权限验证）
- `zongbuRouterMap`：总部路由
- `kuguanRouterMap`：库管路由
- `otherRouterMap`：其他角色路由

### 3. 权限控制 (`src/permission.js`)
