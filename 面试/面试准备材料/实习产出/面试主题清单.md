# 校招面试拷打主题清单

## 你已想到的 4 个方向
1. **业务场景** — 为什么做、给谁用、解决什么问题  
2. **表设计** — 字段、索引、默认值、历史数据兼容  
3. **技术选型** — 为什么用这个技术、有无对比  
4. **代码实现** — 分层、关键逻辑、复用与扩展  

## 补充的 8 个方向
5. **难点与坑点** — 遇到的问题、排查思路、最终方案（最能体现你动手能力）  
6. **性能与扩展** — 异步、批量、大文件、限流/降级思路  
7. **安全与合规** — 脱敏、密码策略、审计、权限  
8. **前后端协作 / 接口设计** — 约定、错误码、文件上传、拦截器  
9. **事务与数据一致性** — 何时用事务、边界、失败怎么处理  
10. **跨系统 / 遗留系统** — .NET + Java 双后端、适配、调用方式  
11. **测试与验证** — 怎么测、边界 case、数据校验  
12. **项目复盘与改进** — 若重做会怎么优化、学到了什么  

## 建议准备顺序
- **必讲熟**：业务场景、表设计、代码实现、难点与坑点（每个功能至少能串成一条线）  
- **常被问**：技术选型、安全与合规、前后端/接口  
- **加分项**：性能与扩展、事务、跨系统、测试、复盘  

详细话术与追问见：**校招面试准备-主题与话术.md**。
