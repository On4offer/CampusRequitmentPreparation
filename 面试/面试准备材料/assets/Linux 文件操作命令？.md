## 面试题：Linux 常用的文件操作命令有哪些？

### 1. 概念

Linux 是一个 **一切皆文件** 的系统，文件操作是最基础也是最常考的部分，掌握文件查看、创建、删除、移动、复制等命令，是面试高频考点。

------

### 2. 常见命令分类

#### 📂 文件查看

- `ls`：列出目录下的文件
- `cat`：查看文件内容（一次性显示）
- `more` / `less`：分页查看大文件
- `head` / `tail`：查看文件头或尾部内容
- `wc`：统计文件的行数、单词数、字节数

#### 📝 文件编辑与创建

- `touch`：新建空文件
- `vi` / `nano`：编辑文件
- `echo` / `printf`：向文件写内容
- `> / >>`：输出重定向（覆盖/追加）

#### 🔧 文件复制与移动

- `cp`：复制文件/目录
- `mv`：移动/重命名文件

#### ❌ 文件删除

- `rm`：删除文件/目录
- `rmdir`：删除空目录

#### 🔍 文件搜索

- `find`：按文件名/条件查找
- `grep`：按内容搜索文件
- `locate`：基于索引快速查找

#### 📦 文件压缩与解压

- `tar`：打包与解包
- `gzip` / `gunzip`：压缩与解压
- `zip` / `unzip`：压缩与解压

------

### 3. 案例演示

```bash
# 查看当前目录下文件（包含详细信息）
ls -l

# 创建文件并写入内容
echo "Hello Linux" > test.txt

# 复制 test.txt 到 backup 目录
cp test.txt ./backup/

# 查找当前目录下所有 .log 文件
find . -name "*.log"

# 实时查看日志文件
tail -f app.log
```

------

### 4. 使用场景

- 日志排查：`tail -f` 实时监控日志
- 大文件查看：`less` 分页浏览
- 自动化脚本：结合 `grep`、`find`、`wc` 批量处理文件
- 系统维护：`rm -rf` 清理临时目录，`tar` 备份项目

------

### 5. 标准回答（面试简洁版）

> Linux 中常用的文件操作命令主要包括：
>
> - **查看类**：`ls`、`cat`、`more`、`less`、`head`、`tail`
> - **编辑/创建类**：`touch`、`echo`、`vi`
> - **移动/复制类**：`cp`、`mv`
> - **删除类**：`rm`、`rmdir`
> - **搜索类**：`find`、`grep`、`locate`
> - **压缩/解压类**：`tar`、`gzip`、`zip`
>    这些命令是日常开发、运维和排错时的基础工具。
