## JMeter 常用命令

### 基本命令

```bash
# 启动图形界面
jmeter

# 命令行模式执行测试
jmeter -n -t test-plan.jmx -l results.jtl -e -o report

# 分布式测试
jmeter -n -t test-plan.jmx -R server1,server2,server3 -l results.jtl

# 查看版本
jmeter --version
```

### 命令参数说明

| 参数 | 说明 |
|------|------|
| `-n` | 非GUI模式 |
| `-t` | 测试计划文件 |
| `-l` | 结果文件 |
| `-e` | 生成报告 |
| `-o` | 报告输出目录 |
| `-R` | 远程服务器列表 |
| `-H` | 代理主机 |
| `-P` | 代理端口 |

### 常用配置

**jmeter.properties**

```properties
# 线程组配置
thread_group.default_threads=100
thread_group.default_ramp_time=10
thread_group.default_duration=60

# HTTP配置
httpclient4.retry_count=3
httpclient4.timeout=30000

# 结果收集
jmeter.save.saveservice.output_format=csv
jmeter.save.saveservice.response_data=true

# 日志配置
log_level.jmeter=INFO
log_level.jmeter.junit=DEBUG
```

**user.properties**

```properties
# 自定义变量
BASE_URL=http://localhost:8080
USERNAME=test
PASSWORD=password

# 计数器配置
counter.start=1
counter.increment=1
counter.maximum=1000
```

## Gatling 常用命令

### 基本命令

```bash
# 启动Gatling
./gatling.sh

# 执行指定模拟
./gatling.sh -s BasicSimulation

# 执行并生成报告
./gatling.sh -s BasicSimulation -ro

# 查看版本
./gatling.sh --version
```

### 命令参数说明

| 参数 | 说明 |
|------|------|
| `-s` | 模拟类名称 |
| `-ro` | 生成报告 |
| `-rd` | 报告描述 |
| `-rf` | 报告文件夹 |
| `-sf` | 模拟文件夹 |
| `-df` | 数据文件夹 |

### 常用配置

**gatling.conf**

```conf
gatling {
  core {
    directory {
      simulations = "user-files/simulations"
      resources = "user-files/resources"
      results = "results"
    }
    run {
      description = "Default run"
      noReports = false
      outputDirectoryBaseName = "gatling"
    }
  }
  http {
    warmUpUrl = ""
    ahc {
      maxConnections = 10000
      maxConnectionsPerHost = 6
      connectTimeout = 60000
      requestTimeout = 60000
    }
  }
}
```

## 排错指南

### 常见错误与解决方案

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| **连接超时** | 网络问题、服务器过载 | 检查网络连接、增加服务器资源 |
| **内存不足** | JVM内存设置过小 | 调整JVM内存参数 |
| **CPU使用率高** | 计算密集型操作 | 优化代码、增加CPU资源 |
| **磁盘I/O高** | 频繁磁盘读写 | 使用缓存、优化文件操作 |
| **数据库连接池耗尽** | 连接数设置不足 | 增加连接池大小 |
| **慢查询** | SQL优化不足 | 优化SQL、添加索引 |
| **线程阻塞** | 同步调用、死锁 | 使用异步处理、解决死锁 |
| **网络延迟** | 网络带宽不足 | 优化网络配置、使用CDN |

### K6 脚本模板

```javascript
import http from 'k6/http';
import { check, sleep } from 'k6';
import { Rate } from 'k6/metrics';

// 自定义指标
let errorRate = new Rate('errors');

export let options = {
  stages: [
    { duration: '1m', target: 10 },
    { duration: '2m', target: 50 },
    { duration: '5m', target: 100 },
    { duration: '2m', target: 50 },
    { duration: '1m', target: 0 },
  ],
  thresholds: {
    http_req_duration: ['p(95)<500'],
    errors: ['rate<0.01'],
  },
};

export function setup() {
  // 初始化数据
  let users = [];
  for (let i = 0; i < 1000; i++) {
    users.push({
      username: `user${i}`,
      password: 'password123',
    });
  }
  return { users };
}

export default function (data) {
  // 随机选择用户
  let user = data.users[Math.floor(Math.random() * data.users.length)];
  
  // 登录
  let loginRes = http.post('http://localhost:8080/api/auth/login', JSON.stringify({
    username: user.username,
    password: user.password,
  }), {
    headers: { 'Content-Type': 'application/json' },
  });
  
  let loginSuccess = check(loginRes, {
    'login status is 200': (r) => r.status === 200,
  });
  
  errorRate.add(!loginSuccess);
  
  if (loginSuccess) {
    let token = loginRes.json('token');
    
    // 浏览商品
    let productsRes = http.get('http://localhost:8080/api/products', {
      headers: { 'Authorization': `Bearer ${token}` },
    });
    
    let productsSuccess = check(productsRes, {
      'products status is 200': (r) => r.status === 200,
    });
    
    errorRate.add(!productsSuccess);
  }
  
  sleep(1);
}

export function teardown(data) {
  // 清理数据
  console.log('Test completed');
}
```

### Gatling 脚本模板

```scala
import io.gatling.core.Predef._
import io.gatling.http.Predef._
import scala.concurrent.duration._

class BasicSimulation extends Simulation {
  
  // HTTP配置
  val httpProtocol = http
    .baseUrl("http://localhost:8080")
    .acceptHeader("application/json")
    .contentTypeHeader("application/json")
  
  // 场景定义
  val scn = scenario("Basic Scenario")
    .exec(http("Get Users")
      .get("/api/users")
      .check(status.is(200)))
    .pause(1, 3)
    .exec(http("Get User Detail")
      .get("/api/users/1")
      .check(status.is(200)))
  
  // 设置
  setUp(
    scn.inject(
      rampUsers(100) during (10 seconds),
      constantUsersPerSec(10) during (5 minutes)
    )
  ).protocols(httpProtocol)
}
```
