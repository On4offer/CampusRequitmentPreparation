## JUnit常用命令

### 基本命令

```bash
# 运行所有测试
mvn test

# 运行特定测试类
mvn test -Dtest=UserServiceTest

# 运行特定测试方法
mvn test -Dtest=UserServiceTest#testRegister

# 运行匹配的测试
mvn test -Dtest=*ServiceTest

# 运行测试并生成报告
mvn test surefire-report:report
```

### 命令参数说明

| 参数 | 说明 |
|------|------|
| `-Dtest` | 指定测试类或测试方法 |
| `-Dgroups` | 按标签分组运行测试 |
| `-DexcludedGroups` | 排除指定分组的测试 |
| `-DskipTests` | 跳过测试 |
| `-Dmaven.test.failure.ignore` | 忽略测试失败 |

### 测试分类运行

```bash
# 运行单元测试
mvn test -Dgroups=unit

# 运行集成测试
mvn test -Dgroups=integration

# 运行E2E测试
mvn test -Dgroups=e2e

# 排除慢速测试
mvn test -DexcludedGroups=slow

# 运行快速测试
mvn test -Dgroups=fast
```

### 常用配置

**pom.xml配置**

```xml
<plugin>
    <groupId>org.apache.maven.plugins</groupId>
    <artifactId>maven-surefire-plugin</artifactId>
    <version>3.0.0-M5</version>
    <configuration>
        <includes>
            <include>**/*Test.java</include>
            <include>**/*Tests.java</include>
        </includes>
        <excludes>
            <exclude>**/IntegrationTest.java</exclude>
            <exclude>**/E2ETest.java</exclude>
        </excludes>
        <parallel>methods</parallel>
        <threadCount>4</threadCount>
        <argLine>-Xmx512m</argLine>
    </configuration>
</plugin>
```

**Gradle配置**

```groovy
test {
    useJUnitPlatform()
    
    // 并行测试
    maxParallelForks = Runtime.runtime.availableProcessors()
    
    // 测试配置
    testLogging {
        events "passed", "skipped", "failed"
        exceptionFormat "full"
        showStandardStreams = false
    }
    
    // JVM参数
    jvmArgs '-Xmx512m'
}
```

## Mockito常用命令

### 基本用法

```java
// 创建Mock对象
UserService userService = Mockito.mock(UserService.class);

// 使用注解创建Mock
@ExtendWith(MockitoExtension.class)
public class UserServiceTest {
    @Mock
    private UserService userService;
}

// 创建Spy对象
List<String> list = new ArrayList<>();
List<String> spyList = Mockito.spy(list);
```

### 设置行为（when-then）

```java
// 设置返回值
when(userService.getUserById(1)).thenReturn(new User("张三"));

// 设置异常
when(userService.getUserById(-1))
    .thenThrow(new IllegalArgumentException("用户ID不能为负数"));

// 设置多次调用返回不同值
when(userService.getUserById(anyInt()))
    .thenReturn(new User("第一次"))
    .thenReturn(new User("第二次"))
    .thenReturn(new User("第三次"));

// 设置连续调用
when(userService.getUserById(anyInt()))
    .thenAnswer(invocation -> {
        Integer id = invocation.getArgument(0);
        return new User("用户" + id);
    });

// Spy对象设置行为
doReturn("李四").when(spyList).get(0);
```

### 验证调用（verify）

```java
// 验证方法被调用
userService.getUserById(1);
verify(userService).getUserById(1);

// 验证方法从未被调用
verify(userService, never()).getUserById(2);

// 验证方法调用次数
verify(userService, times(1)).getUserById(1);
verify(userService, atLeastOnce()).getUserById(1);
verify(userService, atMost(2)).getUserById(1);

// 验证调用顺序
InOrder inOrder = inOrder(userService, orderService);
inOrder.verify(userService).getUserById(1);
inOrder.verify(orderService).createOrder(any());

// 验证超时
verify(userService, timeout(1000)).getUserById(1);
```

### 参数匹配器

```java
// 精确匹配
when(userService.getUserById(1)).thenReturn(user);

// 任意匹配
when(userService.getUserById(anyInt())).thenReturn(user);
when(userService.findUser(anyString())).thenReturn(user);

// 特定条件匹配
when(userService.findUser(contains("张"))).thenReturn(user);
when(userService.findUser(startsWith("李"))).thenReturn(user);
when(userService.findUser(endsWith("王"))).thenReturn(user);
when(userService.findUser(matches(".*三.*"))).thenReturn(user);

// 自定义匹配器
when(userService.findUser(argThat(user -> user.getAge() > 18)))
    .thenReturn(user);

// 多个参数匹配
when(userService.createUser(anyString(), anyInt()))
    .thenReturn(user);

// 精确匹配与任意匹配混用
when(userService.createUser("张三", anyInt()))
    .thenReturn(user);
```

### 高级特性

```java
// 静态方法Mock（Mockito 3.4+）
try (MockedStatic<StringUtils> mocked = mockStatic(StringUtils.class)) {
    mocked.when(() -> StringUtils.isEmpty(anyString())).thenReturn(true);
    assertTrue(StringUtils.isEmpty("test"));
}

// 构造函数Mock（Mockito 3.5+）
try (MockedConstruction<User> mocked = mockConstruction(User.class)) {
    User user = new User("张三");
    when(user.getName()).thenReturn("模拟用户");
    assertEquals("模拟用户", user.getName());
}

// 重置Mock
Mockito.reset(userService);

// 验证无更多交互
verifyNoMoreInteractions(userService);

// 验证零交互
verifyNoInteractions(userService);
```

## AssertJ常用命令

### 基本断言

```java
// 对象断言
assertThat(user)
    .isNotNull()
    .isInstanceOf(User.class)
    .isEqualTo(expectedUser)
    .isNotEqualTo(otherUser);

// 数值断言
assertThat(age)
    .isEqualTo(25)
    .isGreaterThan(18)
    .isLessThan(30)
    .isBetween(20, 30)
    .isPositive()
    .isNotNegative();

// 布尔断言
assertThat(isValid)
    .isTrue()
    .isFalse();

// 字符串断言
assertThat(name)
    .isNotEmpty()
    .startsWith("张")
    .endsWith("三")
    .contains("张三")
    .hasSize(2)
    .isEqualTo("张三");
```

### 集合断言

```java
// List断言
assertThat(users)
    .isNotEmpty()
    .hasSize(3)
    .contains(user1, user2)
    .containsExactly(user1, user2, user3)
    .doesNotContain(user4)
    .hasOnlyElementsOfType(User.class);

// 提取属性断言
assertThat(users)
    .extracting("name")
    .contains("张三", "李四", "王五");

// 过滤断言
assertThat(users)
    .filteredOn("age", 25)
    .hasSize(2);

// Map断言
assertThat(map)
    .isNotEmpty()
    .hasSize(3)
    .containsKey("name")
    .containsValue("张三")
    .containsEntry("name", "张三");
```

### 异常断言

```java
// 断言抛出异常
assertThatThrownBy(() -> calculator.divide(10, 0))
    .isInstanceOf(ArithmeticException.class)
    .hasMessage("/ by zero")
    .hasNoCause();

// 断言异常链
assertThatThrownBy(() -> userService.getUserById(-1))
    .isInstanceOf(BusinessException.class)
    .hasMessage("用户不存在")
    .hasCauseInstanceOf(NotFoundException.class);

// 断言不抛出异常
assertThatCode(() -> userService.register(request))
    .doesNotThrowAnyException();
```

### 对象属性断言

```java
// 断言对象属性
assertThat(user)
    .hasFieldOrProperty("name")
    .hasFieldOrPropertyWithValue("name", "张三")
    .hasNoNullFieldsOrProperties();

// 使用提取器
assertThat(user)
    .extracting("name", "age", "email")
    .containsExactly("张三", 25, "zhangsan@example.com");

// 比较对象
assertThat(actualUser)
    .usingRecursiveComparison()
    .isEqualTo(expectedUser);

// 忽略字段比较
assertThat(actualUser)
    .usingRecursiveComparison()
    .ignoringFields("id", "createTime")
    .isEqualTo(expectedUser);
```

## 常见测试场景

### 1. 测试正常流程

```java
@Test
public void testRegister_Success() {
    // Arrange
    RegisterRequest request = new RegisterRequest("zhangsan@example.com", "password123");
    User savedUser = new User(1L, "zhangsan@example.com", "password123");
    
    when(userRepository.existsByEmail(anyString())).thenReturn(false);
    when(userRepository.save(any(User.class))).thenReturn(savedUser);
    
    // Act
    User result = userService.register(request);
    
    // Assert
    assertThat(result)
        .isNotNull()
        .hasFieldOrPropertyWithValue("id", 1L)
        .hasFieldOrPropertyWithValue("email", "zhangsan@example.com");
    
    verify(userRepository).existsByEmail("zhangsan@example.com");
    verify(userRepository).save(any(User.class));
}
```

### 2. 测试异常流程

@Test

    // Act & Assert

}

## 排错指南

### 常见错误与解决方案

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| **Mock对象返回null** | 没有设置Mock行为 | 使用when().thenReturn()设置返回值 |
| **测试失败：Expected exception not thrown** | 预期的异常没有抛出 | 检查测试条件，确保会抛出异常 |
| **测试超时** | 测试执行时间过长 | 优化测试代码，使用Mock隔离外部依赖 |
| **Mock对象验证失败** | 方法调用次数或参数不匹配 | 检查verify语句，确保调用次数和参数正确 |
| **测试覆盖率不足** | 测试用例覆盖不全 | 补充测试用例，特别是边界条件和异常场景 |
| **测试依赖外部资源** | 测试依赖数据库、网络等 | 使用Mock或Testcontainers隔离外部依赖 |
| **测试不稳定** | 测试结果不一致 | 检查测试数据，确保测试独立和可重复 |
