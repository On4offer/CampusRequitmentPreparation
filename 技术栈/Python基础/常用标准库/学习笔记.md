# 📘《Python 常用标准库学习与实战手册》

> 系统学习见下文各章；日常常用 API 与场景速查见同目录《常用API与使用场景》。

------

## 第1章：标准库概述

### 1.1 什么是标准库

- **标准库**是随 Python 安装的模块与包，无需 pip 安装即可 import；涵盖文件、正则、日期、数据结构、并发、网络等。
- 常用分类：**内置函数与类型**（len、range、open）、**数据类型**（collections、json）、**文件与路径**（os、pathlib）、**文本与正则**（re）、**时间**（datetime、time）、**网络与 I/O**（urllib、socket）、**并发**（threading、multiprocessing）。

### 1.2 文档与查找

- 官方文档 [docs.python.org](https://docs.python.org/)；help(模块/函数)、dir(模块) 交互查看；第三方库用 pip install 后同理。

## 第2章：文件与路径

1. **open**：open(path, mode='r', encoding='utf-8')；模式 r/w/a、rb/wb、r+/w+；**with open(...) as f** 自动关闭；f.read()、f.readline()、f.readlines()、f.write()、f.writelines()；迭代行 for line in f。
2. **pathlib.Path**：面向对象路径；Path('a/b/c')、p.exists()、p.is_file()、p.read_text()、p.write_text()、p.iterdir()、p.glob('*.py')；跨平台分隔符。
3. **os**：os.path.join、os.path.exists、os.listdir、os.mkdir、os.getcwd、os.chdir、os.environ 环境变量。
4. **面试要点**：with 保证关闭、encoding 与二进制模式、pathlib 与 os.path 取舍。

------

## 第3章：正则表达式（re）

1. **re 模块**：re.search(pattern, string) 找首个匹配、re.match 从开头匹配、re.findall 找全部、re.sub 替换、re.split 按模式切分。
2. **匹配对象**：m.group()、m.groups()、m.start()、m.end()；分组 () 与命名 (?P&lt;name&gt;...)。
3. **模式**：\d、\w、\s、.、*、+、?、[]、^、$、|；(?:...) 非捕获组；re.IGNORECASE、re.DOTALL 等标志。
4. **编译**：re.compile(pattern) 多次匹配时复用可提效。
5. **面试要点**：search 与 match 区别、贪婪与非贪婪（*?）、常用元字符。

------

## 第4章：json 与数据序列化

1. **json**：json.dumps(obj) 对象→字符串、json.loads(s) 字符串→对象；json.dump(obj, fp)、json.load(fp) 写读文件。
2. **类型对应**：dict↔object、list↔array、str↔string、int/float↔number、True/False↔true/false、None↔null；自定义类型需 default 与 object_hook 或继承 JSONEncoder。
3. **参数**：ensure_ascii=False 保留中文、indent 美化输出、sort_keys 键排序。
4. **面试要点**：dumps 与 dump 区别、不能直接序列化的类型如何处理。

------

## 第5章：日期与时间（datetime）

1. **datetime**：datetime.datetime.now()、datetime.date.today()；构造 datetime(year, month, day, hour, minute, second)；**timedelta** 表示时间差，用于加减。
2. **格式化**：strftime('%Y-%m-%d %H:%M:%S') 输出字符串；strptime(s, format) 解析字符串为 datetime。
3. **时间戳**：.timestamp() 转时间戳、datetime.fromtimestamp(ts) 从时间戳转回；与系统时区相关。
4. **面试要点**：strftime 与 strptime、timedelta 计算、注意时区（可用 zoneinfo 或第三方 pytz）。

------

## 第6章：collections 与 itertools

1. **collections**：**defaultdict** 默认值字典、**Counter** 计数（most_common）、**deque** 双端队列（popleft、appendleft）、**namedtuple** 命名元组、**OrderedDict** 有序字典（3.7+ dict 已保序）。
2. **itertools**：count、cycle、chain、islice；permutations、combinations；groupby；常用于惰性迭代与组合枚举。
3. **面试要点**：Counter 统计词频、deque 做队列或滑动窗口、defaultdict 避免 KeyError。

------

## 第7章：常用内置与其它

1. **内置**：len、range、enumerate、zip、map、filter、sorted、reversed、any、all、sum、min、max；open、print、input；isinstance、hasattr、getattr。
2. **math**：ceil、floor、sqrt、log、pow；**random**：random.random()、random.randint(a,b)、random.choice(lst)、random.shuffle(lst)。
3. **sys**：sys.argv、sys.path、sys.exit()；**functools**：reduce、lru_cache、partial、wraps。
4. **面试要点**：map/filter 与列表推导取舍、reduce 用途、lru_cache 做缓存。
