> 系统学习见下文各章；日常常用语法与 API 速查见同目录《常用API与使用场景》。

## 第1章：Python 概述

### 1.1 Python 简介

- **Python** 是一种**解释型、动态类型**的高级语言，强调可读性与简洁（“优雅胜于丑陋”）；广泛应用于 Web、数据分析、自动化、AI/ML。
- 主流版本 **Python 3**；解释器 CPython 常用；虚拟环境（venv、conda）用于隔离项目依赖。

### 1.2 运行与开发环境

- 安装 Python、pip 包管理；**venv**：python -m venv .venv、激活后 pip install；IDE/编辑器（VS Code、PyCharm）与运行配置。
- 交互式 REPL、脚本 .py 执行、模块导入；__name__ == "__main__" 区分直接运行与导入。

## 第2章：基础语法与类型

1. **变量与命名**：无需声明类型，赋值即定义；命名规范（蛇形、常量全大写）；标识符规则。
2. **数据类型**：**数值**（int、float、complex）、**序列**（str、list、tuple、range）、**映射**（dict）、**集合**（set、frozenset）、**布尔**（bool）、**None**。
3. **类型判断与转换**：type()、isinstance()；int()、float()、str()、list()、dict() 等转换。
4. **运算符**：算术、比较、逻辑（and、or、not）、身份（is、is not）、成员（in、not in）；运算符优先级。
5. **字符串**：单/双/三引号、转义、原始字符串 r"..."；拼接（+、join）、切片、常用方法（strip、split、replace、format、f-string）。

------

## 第3章：控制流

1. **条件**：if / elif / else；条件表达式（三元）x if cond else y；真值测试（非零、非空容器为 True）。
2. **循环**：for ... in ...（可遍历序列、range、字典键值）；while；break、continue；else 子句（循环正常结束执行）。
3. **range**：range(n)、range(start, stop, step)；常用于 for 计数与索引。
4. **列表推导式**：[x*2 for x in lst]、带条件 [x for x in lst if x>0]；字典与集合推导式。

------

## 第4章：函数

1. **定义**：def 函数名(参数): 函数体；无 return 则返回 None。
2. **参数**：位置参数、默认参数（默认值只求一次，可变对象需注意）；*args 可变位置、**kwargs 可变关键字；参数顺序：位置 → *args → 默认 → **kwargs。
3. **返回值**：可多值返回（实际为元组解包）；提前 return。
4. **作用域**：LEGB（Local、Enclosing、global、Built-in）；global、nonlocal 声明。
5. **lambda**：匿名函数，单表达式；常用于 sorted、map、filter 的 key 或函数参数。
6. **文档与类型注解**："""docstring"""；可选类型注解（如 def f(x: int) -> str:），不影响运行，便于 IDE 与静态检查。

------

## 第5章：数据结构进阶

1. **列表 list**：可变、有序；索引与切片、append、extend、insert、remove、pop、sort、reverse；列表推导式。
2. **元组 tuple**：不可变、有序；拆包、命名元组（collections.namedtuple）；单元素元组 (x,)。
3. **字典 dict**：键值对、键唯一且不可变；dict()、{}、dict.fromkeys；get、keys、values、items；更新与合并（update、|）。
4. **集合 set**：无序、不重复；交并差（&、|、-）；去重、成员检测 O(1)；frozenset 不可变。
5. **切片**：lst[start:stop:step]、负索引、浅拷贝；字符串与列表通用。

------

## 第6章：模块与包

1. **模块**：一个 .py 文件即模块；import 模块名、from 模块 import 成员；__name__ 与 __main__。
2. **包**：含 __init__.py 的目录；import 包.子模块；__init__.py 可空或做包初始化。
3. **常用内置**：math、random、os、sys、pathlib；第三方用 pip 安装后 import。
4. **面试要点**：模块搜索路径、if __name__ == "__main__" 作用、相对导入与绝对导入。
