## 🏗️ 面向对象速查

### 类与对象

```java
// 定义类
public class Person {
    // 成员变量（属性）
    private String name;
    private int age;
    
    // 构造方法
    public Person() {}                        // 无参构造
    
    public Person(String name, int age) {     // 有参构造
        this.name = name;
        this.age = age;
    }
    
    // Getter/Setter
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
    
    // 普通方法
    public void sayHello() {
        System.out.println("Hello, I'm " + name);
    }
    
    // 重写 toString
    @Override
    public String toString() {
        return "Person{name='" + name + "', age=" + age + "}";
    }
}

// 创建对象
Person p1 = new Person();
Person p2 = new Person("张三", 20);

// 使用对象
p2.sayHello();
System.out.println(p2.getName());
```

### 继承

```java
// 父类
public class Animal {
    protected String name;
    
    public void eat() {
        System.out.println(name + " is eating");
    }
}

// 子类
public class Dog extends Animal {
    public Dog(String name) {
        this.name = name;
    }
    
    // 重写父类方法
    @Override
    public void eat() {
        System.out.println(name + " is eating bones");
    }
    
    // 子类特有方法
    public void bark() {
        System.out.println("Woof!");
    }
}

// 使用
Animal animal = new Dog("Buddy");   // 向上转型
animal.eat();                        // 调用重写的方法
// animal.bark();                    // ✗ 编译错误，Animal 类型没有 bark 方法

Dog dog = (Dog) animal;              // 向下转型（需确保类型正确）
dog.bark();                          // ✓ 可以调用
```

### 抽象类与接口

```java
// 抽象类
public abstract class Shape {
    protected String color;
    
    public Shape(String color) {
        this.color = color;
    }
    
    // 抽象方法（子类必须实现）
    public abstract double getArea();
    
    // 普通方法
    public void printColor() {
        System.out.println("Color: " + color);
    }
}

// 接口
public interface Drawable {
    void draw();                    // 默认 public abstract
    
    // 默认方法（JDK 8+）
    default void printInfo() {
        System.out.println("This is drawable");
    }
    
    // 静态方法（JDK 8+）
    static void showType() {
        System.out.println("Interface Drawable");
    }
}

// 实现
public class Circle extends Shape implements Drawable {
    private double radius;
    
    public Circle(String color, double radius) {
        super(color);
        this.radius = radius;
    }
    
    @Override
    public double getArea() {
        return Math.PI * radius * radius;
    }
    
    @Override
    public void draw() {
        System.out.println("Drawing a circle");
    }
}
```

## 🎯 常用代码场景

### 1. 单例模式（饿汉式）

```java
public class Singleton {
    private static final Singleton INSTANCE = new Singleton();
    
    private Singleton() {}                  // 私有构造
    
    public static Singleton getInstance() {
        return INSTANCE;
    }
}
```

### 2. 单例模式（懒汉式 + 双重检查）

```java
public class Singleton {
    private static volatile Singleton instance;
    
    private Singleton() {}
    
    public static Singleton getInstance() {
        if (instance == null) {
            synchronized (Singleton.class) {
                if (instance == null) {
                    instance = new Singleton();
                }
            }
        }
        return instance;
    }
}
```

### 3. 工具类（静态方法）

```java
public final class StringUtils {
    private StringUtils() {}                // 私有构造，防止实例化
    
    public static boolean isEmpty(String str) {
        return str == null || str.trim().isEmpty();
    }
    
    public static String defaultIfEmpty(String str, String defaultStr) {
        return isEmpty(str) ? defaultStr : str;
    }
}

// 使用
String result = StringUtils.defaultIfEmpty(input, "default");
```

### 4. 枚举类

```java
public enum Status {
    SUCCESS(200, "成功"),
    ERROR(500, "错误"),
    NOT_FOUND(404, "未找到");
    
    private final int code;
    private final String message;
    
    Status(int code, String message) {
        this.code = code;

    }

    public String getMessage() { return message; }
```

### 5. 内部类

```java
public class Outer {
    private String outerField = "outer";
    
    // 成员内部类
    public class Inner {
        public void print() {
            System.out.println(outerField); // 可以访问外部类成员
        }
    }
    
    // 静态内部类
    public static class StaticInner {
        public void print() {
            // System.out.println(outerField); // ✗ 不能访问非静态成员
            System.out.println("static inner");
        }
    }
    
    // 局部内部类
    public void method() {
        class LocalInner {
            public void print() {
                System.out.println(outerField);
            }
        }
        new LocalInner().print();
    }
}

// 使用
Outer.Inner inner = new Outer().new Inner();    // 成员内部类
Outer.StaticInner staticInner = new Outer.StaticInner();  // 静态内部类
```

## ⚠️ 常见坑点速查

| 坑点 | 说明 | 正确做法 |
|------|------|----------|
| `==` vs `equals` | `==` 比较引用地址，`equals` 比较内容 | 字符串比较用 `equals` |
| 字符串拼接 | 循环中用 `+` 拼接效率低 | 用 `StringBuilder` |
| 空指针异常 | 调用 null 对象的方法 | 先判空或使用 `Optional` |
| 数组越界 | 访问 `arr[arr.length]` | 注意索引范围 `0 ~ length-1` |
| 整数除法 | `5 / 2 = 2` 不是 `2.5` | 先转 `double`：`5.0 / 2` |
| 自动装箱拆箱 | `Integer` 与 `int` 混用可能有 NPE | 注意 null 值判断 |
| 方法重写 | 参数不同是重载不是重写 | 用 `@Override` 检查 |
| 构造方法继承 | 子类不继承父类构造 | 用 `super()` 调用 |

## 📝 面试代码手写题

### 1. 反转字符串

```java
public String reverse(String s) {
    return new StringBuilder(s).reverse().toString();
}

// 或手动实现
public String reverseManual(String s) {
    char[] chars = s.toCharArray();
    int left = 0, right = chars.length - 1;
    while (left < right) {
        char temp = chars[left];
        chars[left] = chars[right];
        chars[right] = temp;
        left++;
        right--;
    }
    return new String(chars);
}
```

### 2. 判断回文

```java
public boolean isPalindrome(String s) {
    int left = 0, right = s.length() - 1;
    while (left < right) {
        if (s.charAt(left) != s.charAt(right)) {
            return false;
        }
        left++;
        right--;
    }
    return true;
}
```

### 3. 二分查找

```java
public int binarySearch(int[] arr, int target) {
    int left = 0, right = arr.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;  // 防止溢出
        if (arr[mid] == target) {
            return mid;
        } else if (arr[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    return -1;
}
```
