# 🧱 第1章：Python 中的 Redis 使用概述

------

## **1.1 常用客户端库**

### 🧩 背景

- Python 连接 Redis 常用 **redis-py**（官方推荐），提供同步接口；**redis-py** 4.2+ 支持异步（基于 asyncio）。
- 其他可选：**aioredis**（已并入 redis-py 异步）、**hiredis**（C 解析器，加速）。

------

### 🔹 安装与连接

| 库 | 说明 |
|------|------|
| `pip install redis` | 同步 + 异步（4.2+） |
| `pip install hiredis` | 可选，解析性能更好 |

```python
import redis
r = redis.Redis(host="localhost", port=6379, db=0, decode_responses=True)
```

------

## **1.2 同步与异步**

| 模式 | 说明 | 适用 |
|------|------|------|
| `redis.Redis()` | 同步连接，直接 `r.get()` | Flask、脚本 |
| `redis.asyncio.Redis()` | 异步连接，`await r.get()` | FastAPI、asyncio 项目 |

------

### 🧠 关键词总结

> Python 侧使用 **redis-py** 通过 Redis 协议与 Redis 服务通信；同步用 `redis.Redis`，异步用 `redis.asyncio.Redis`，与 Java 的 Jedis/Lettuce 可类比。

------

### 📘 面试延展题

**问：Redis 在 Python 里一般用来做什么？**

**答：** 缓存接口结果或热点数据、分布式锁（set NX EX）、限流（计数器/滑动窗口）、Session 存储、Celery 的 Broker/Backend 等。
