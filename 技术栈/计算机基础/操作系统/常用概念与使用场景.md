# 📌 操作系统常用概念与使用场景速查

> 配合《学习笔记》使用，侧重**进程线程、内存、死锁、调度**，便于速查与校招面试。

---

## 一、进程与线程

| 对比项 | 进程 | 线程 |
|--------|------|------|
| **定义** | 资源分配单位，独立地址空间 | CPU 调度单位，共享进程资源 |
| **切换** | 开销大（切换地址空间、页表等） | 开销小（共享地址空间） |
| **通信** | IPC（管道、消息队列、共享内存、Socket） | 共享全局变量（需同步） |
| **隔离** | 进程间隔离，崩溃不影响其他进程 | 同进程内线程崩溃可能拖垮进程 |

- **用户线程与内核线程**：用户线程在用户态、内核不感知，多对一映射到内核线程；内核线程由 OS 调度。Java 主流实现（HotSpot）是**一对一**映射到内核线程。

## 二、进程状态与调度算法

| 状态 | 说明 |
|------|------|
| **就绪** | 等 CPU |
| **运行** | 正在执行 |
| **阻塞** | 等 I/O 或事件 |

| 调度算法 | 思路 | 特点 |
|----------|------|------|
| **FCFS** | 先来先服务 | 简单；护航效应 |
| **SJF** | 短作业优先 | 平均等待时间短；长作业可能饿死 |
| **时间片轮转 RR** | 固定时间片轮流 | 公平、响应快；时间片大小影响切换开销 |
| **多级反馈队列** | 多级队列、不同时间片、可升降级 | 综合长短作业、交互式；**常用** |
| **优先级** | 按优先级调度 | 可抢占或非抢占；可能饿死低优先级 |

## 三、内存管理

| 概念 | 说明 |
|------|------|
| **分页** | 逻辑地址按页、物理按帧，页表映射；解决外部碎片；缺页则页面置换 |
| **分段** | 按逻辑段（代码/数据/栈）；有内部碎片；可与分页结合为段页式 |
| **虚拟内存** | 进程地址空间可大于物理内存；按需调入、置换；多道程序提高利用率 |
| **页表** | 虚页号→物理帧号；多级页表节省空间；TLB 加速翻译 |

| 置换算法 | 思路 | 特点 |
|----------|------|------|
| **FIFO** | 先进入的页先换出 | 可能 Belady 异常 |
| **LRU** | 最近最久未用换出 | 效果好；实现成本高（链表+哈希或近似） |
| **Clock** | 环形+引用位，近似 LRU | 实现简单 |
| **LFU** | 最不经常使用 | 可能历史热点长期占用 |

## 四、死锁

| 四个条件（同时满足才可能死锁） | 说明 |
|-------------------------------|------|
| **互斥** | 资源不能共享 |
| **占有且等待** | 持有资源并等待其他资源 |
| **不可剥夺** | 资源不能被强制剥夺 |
| **循环等待** | 存在资源等待环 |

| 策略 | 做法 |
|------|------|
| **预防** | 破坏其一：如一次性申请全部、资源有序编号（破坏循环等待） |
| **避免** | 银行家算法：分配前检测是否进入不安全状态 |
| **检测与恢复** | 允许发生，定期检测（资源分配图）并解除（剥夺、回滚） |

## 五、同步与互斥

| 机制 | 说明 |
|------|------|
| **信号量** | 整型+PV 操作；可表示资源数或互斥（二元信号量） |
| **互斥量 Mutex** | 互斥锁，同一时刻仅一个线程持有 |
| **条件变量** | 配合互斥量，满足条件时唤醒等待线程；生产者-消费者常用 |
| **管程** | 封装互斥与条件变量的高级结构 |

- **生产者-消费者**：有界缓冲区；空时消费者等、满时生产者等；互斥访问缓冲区。
- **读者-写者**：多读者可并发、写者独占；可偏向读者或写者。

## 六、用户态与内核态

| 概念 | 说明 |
|------|------|
| **用户态** | 应用代码执行，权限受限，不能直接访问硬件与敏感资源 |
| **内核态** | 内核代码执行，可访问全部资源 |
| **系统调用** | 用户态通过软中断/陷阱进入内核，执行完返回用户态 |
| **切换代价** | 用户态↔内核态有上下文保存与恢复、权限切换 |

- **为什么区分**：安全与稳定，防止用户程序破坏系统；敏感操作（I/O、内存映射）只能在内核完成。

---

## 七、面试一句话总结

- **进程**：资源分配单位、独立地址空间；**线程**：调度单位、共享进程资源；切换与通信方式不同。
- **调度**：FCFS、SJF、RR、多级反馈队列；多级反馈兼顾长短作业与交互。
- **虚拟内存**：按需调入、页面置换；LRU、Clock 常用；缺页与抖动。
- **死锁**：四条件；预防（破坏其一）、避免（银行家）、检测恢复。
- **同步**：信号量、互斥量、条件变量；生产者-消费者、读者-写者。
- **用户态/内核态**：权限不同；系统调用进入内核；切换有开销。
