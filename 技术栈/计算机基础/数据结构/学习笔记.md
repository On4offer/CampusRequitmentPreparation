# 📘《数据结构学习与实战手册》

> 系统学习见下文各章；日常常用概念与使用场景速查见同目录《常用概念与使用场景》。

------

## 第1章：数据结构概述

### 1.1 什么是数据结构

- **数据结构**是数据的**组织、存储与操作**方式，直接影响算法的效率；常与**算法**一起考察（时间与空间复杂度、稳定性、适用场景）。
- 分类：**线性**（数组、链表、栈、队列）、**树**（二叉树、BST、AVL、红黑树、B 树）、**图**、**散列**（哈希表）。

### 1.2 复杂度分析

- **时间复杂度**：基本操作次数随规模 n 的增长趋势；O(1)、O(log n)、O(n)、O(n log n)、O(n²)、O(2^n) 等；关注最坏与平均。
- **空间复杂度**：额外空间随 n 的增长；递归栈也算。
- **稳定性**（排序）：相等元素相对顺序是否改变；稳定排序在多次排序时有用。

## 第2章：线性表

1. **数组**：连续存储、随机访问 O(1)、插入删除需移动元素 O(n)；适合查多改少、下标访问。
2. **链表**：结点串成链、非连续；单链表、双链表、循环链表；插入删除（已知前驱）O(1)、查找 O(n)；适合频繁插入删除、头尾操作。
3. **数组与链表对比**：访问、插入删除、缓存友好性、扩容；Java 中 ArrayList 与 LinkedList 的取舍。
4. **面试要点**：反转链表、合并有序链表、找环、删除节点（O(1) 换值删后继等技巧）。

------

## 第3章：栈与队列

1. **栈**：LIFO； push、pop、peek；应用：括号匹配、表达式求值、DFS、递归转非递归、单调栈（ next greater element）。
2. **队列**：FIFO；enqueue、dequeue；应用：BFS、滑动窗口、任务调度、消息队列抽象。
3. **双端队列 Deque**：两端可入队出队；可做栈或队列；滑动窗口最值常用单调队列。
4. **优先队列**：按优先级出队；堆实现；TopK、合并 K 个有序链表、Dijkstra。
5. **面试要点**：用栈实现队列/用队列实现栈、最小栈、单调栈与单调队列应用。

------

## 第4章：树与二叉树

1. **树**：层次结构、根、子节点、叶子、深度与高度；二叉树每个节点最多两子。
2. **遍历**：前序、中序、后序（递归与迭代）、层序；中序+二叉搜索树得到有序序列。
3. **二叉搜索树 BST**：左小右大；查找、插入、删除 O(h)，h 为高；若退化成链则 O(n)。
4. **平衡二叉树**：AVL（任意节点左右子树高度差≤1）、红黑树（近似平衡、旋转少）；保证 O(log n) 操作。
5. **红黑树**：五大性质、左旋右旋、插入与删除的调整；Java TreeMap、HashMap 桶内链表转红黑树。
6. **面试要点**：前中后序与层序、BST 与平衡、红黑树特点、最近公共祖先、树的直径与路径。

------

## 第5章：堆与并查集

1. **堆**：完全二叉树、父节点与子节点满足大小关系；大顶堆、小顶堆；插入与删除堆顶 O(log n)、取最值 O(1)。
2. **应用**：优先队列、堆排序、TopK（维护大小为 K 的小顶堆）、中位数（双堆）。
3. **并查集**：维护不相交集合；find（带路径压缩）、union（按秩合并）；判环、连通分量、最小生成树 Kruskal。
4. **面试要点**：堆的插入与删除过程、TopK 两种思路（堆 vs 快排划分）、并查集优化与复杂度。

------

## 第6章：图

1. **图的表示**：邻接矩阵（稠密）、邻接表（稀疏）；有权与无权。
2. **遍历**：BFS（层序、最短路径边权为 1）、DFS（路径、连通分量、拓扑排序基础）。
3. **最短路径**：BFS（无权）、Dijkstra（非负权、贪心）、Bellman-Ford（可有负权）、Floyd（多源、任意两点）。
4. **拓扑排序**：有向无环图 DAG 的线性序；入度为 0 的节点依次移除或 DFS 逆序；依赖排序、编译顺序。
5. **面试要点**：BFS/DFS 区别、Dijkstra 思想、拓扑排序实现、判断有向图是否有环。

------

## 第7章：哈希与字符串

1. **哈希表**：键通过哈希函数映射到下标；冲突解决：链地址法、开放寻址（线性探测、二次探测）；平均 O(1) 查找与插入；负载因子与扩容。
2. **应用**：去重、计数、两数之和、子串统计；Java HashMap、HashSet。
3. **字符串**：KMP（模式匹配、next 数组）、滑动窗口、前缀与后缀；常见题：最长无重复子串、回文、子串匹配。
4. **面试要点**：哈希冲突解决、HashMap 原理简述、KMP 思想、滑动窗口模板。

------

## 第8章：排序与查找

1. **排序**：冒泡、选择、插入（O(n²)）；归并、快排（O(n log n)）；堆排序；桶排序、计数排序、基数排序（线性条件特殊）；稳定性与适用场景。
2. **二分查找**：有序序列 O(log n)；二分答案（最大值最小化）；边界与死循环注意。
3. **面试要点**：快排与归并思想、稳定性、手写快排与二分、TopK 与排序的关系。
