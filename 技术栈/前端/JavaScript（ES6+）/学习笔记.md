## 第1章：JavaScript 概述与运行环境

### 1.1 JavaScript 简介

- **JavaScript** 是一种**单线程、动态类型**的脚本语言，运行在浏览器与 Node.js 等环境中，负责页面的交互与逻辑。
- **ES6（ES2015）及后续** 带来 let/const、箭头函数、Promise、模块化、解构、类等，是现代前端与校招笔试/面试的重点。

### 1.2 运行环境与执行机制

- **浏览器**：JS 引擎（如 V8）、DOM/BOM API、事件循环（Event Loop）；**Node.js**：无 DOM，提供 fs、http 等模块。
- **单线程与异步**：主线程执行同步代码；异步任务（定时器、Promise、I/O）通过任务队列与事件循环在适当时机执行。
