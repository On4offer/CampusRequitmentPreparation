# 📘《TypeScript 学习与实战手册》

> 系统学习见下文各章；日常常用语法与类型速查见同目录《常用API与使用场景》。

------

## 第1章：TypeScript 概述

### 1.1 TypeScript 简介

- **TypeScript** 是 JavaScript 的**超集**，在 JS 基础上增加**静态类型系统**，编译为 JS 后运行于浏览器或 Node。
- 优势：类型检查减少运行时错误、编辑器智能提示与重构、便于大型项目与团队协作；Vue 3、React 生态普遍支持 TS。

### 1.2 环境与配置

- 安装：`npm i -D typescript`；编译：`tsc` 或通过 Vite/Webpack 集成。
- **tsconfig.json**：target、module、strict、paths、include/exclude 等；与前端工程化结合（Vite 自带 TS 支持）。

## 第2章：基础类型与推断

1. 基本类型：number、string、boolean、null、undefined、symbol、bigint
2. 数组与元组：`type[]`、`Array<type>`、`[string, number]`
3. 字面量类型与联合类型：`'a' | 'b'`、`1 | 2`
4. 类型推断：省略注解时 TS 根据赋值推断；any 与关闭检查
5. 类型断言：`as Type`、`<Type>value`；非空断言 `!`

------

## 第3章：对象、接口与类型别名

1. 接口 interface：可选属性、只读、索引签名、继承
2. 类型别名 type：联合、交叉（&）、与 interface 的异同
3. 常用内置类型：Partial、Required、Readonly、Pick、Omit、Record
4. 泛型入门：`function fn<T>(x: T): T`、泛型约束 extends

------

## 第4章：函数与泛型

1. 函数类型：参数类型、返回值类型、可选参数与默认值、剩余参数
2. 函数重载： overload 声明 + 实现签名
3. 泛型函数与泛型接口：`<T>` 在函数与类型上的使用
4. 泛型约束：`T extends { length: number }`、keyof、在类型中引用自身

------

## 第5章：类与进阶类型

1. 类的类型：属性、构造函数、readonly、访问修饰符（public/private/protected）
2. 抽象类与 implements：abstract class、implements interface
3. 高级类型：条件类型、infer、映射类型、模板字面量类型简介
4. 声明文件：.d.ts、declare、为 JS 库写类型、@types/*

------

## 第6章：与 Vue/React 结合

1. Vue 3：`defineComponent`、ref/reactive 类型、Props 与 Emits 类型、组件实例类型
2. React：FC、useState/useRef 泛型、事件类型、Props 与 Hooks 类型
3. 常见坑：严格模式下 undefined 检查、类型断言时机、第三方库类型安装
4. 面试要点：TS 与 JS 对比、interface 与 type、泛型应用、strict 模式意义
