## HTML 常用技巧

### 语义化标签使用场景

- **页面结构**：使用 `<header>`、`<nav>`、`<main>`、`<section>`、`<article>`、`<aside>`、`<footer>` 构建清晰的页面结构
- **表单**：使用 `<label for="input-id">` 关联标签与输入框，提高可访问性
- **媒体内容**：使用 `<figure>` 和 `<figcaption>` 组合图片与说明文字
- **列表**：根据内容类型选择 `<ul>`（无序列表）或 `<ol>`（有序列表）

### 表单处理技巧

- **输入验证**：使用 HTML5 内置验证属性，如 `required`、`pattern`、`min`、`max`
- **输入类型**：根据内容选择合适的输入类型，如 `email`、`tel`、`number`、`date`
- **文件上传**：使用 `<input type="file">` 支持文件选择，添加 `multiple` 属性支持多文件
- **表单提交**：使用 `form` 的 `action` 和 `method` 属性，或通过 JavaScript 处理提交

### 响应式设计基础

- **视口设置**：在 `<head>` 中添加 `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
- **图片响应式**：使用 `max-width: 100%; height: auto;` 确保图片适应容器
- **媒体查询**：使用 `@media` 规则根据屏幕尺寸调整样式

## CSS 常用技巧

### 布局技巧

- **Flexbox 布局**：使用 `display: flex` 实现灵活的一维布局，适用于导航栏、卡片网格等
- **Grid 布局**：使用 `display: grid` 实现二维布局，适用于复杂页面结构
- **盒模型**：理解 `content-box` vs `border-box`，推荐使用 `box-sizing: border-box`
- **定位**：使用 `position: relative/absolute/fixed/sticky` 实现元素定位

### 样式优化

- **CSS 变量**：使用 `--variable-name` 定义变量，提高代码可维护性
- **选择器优化**：避免过度嵌套，使用语义化类名
- **动画效果**：使用 `transition` 和 `transform` 实现平滑过渡，避免使用 `left`、`top` 等属性
- **字体优化**：使用 `font-family` 回退机制，考虑字体加载性能

### 常用场景解决方案

- **居中对齐**：Flexbox 或 Grid 实现水平垂直居中
- **清除浮动**：使用 `clear: both` 或 `overflow: hidden`
- **响应式导航**：在小屏幕上转为汉堡菜单
- **图片懒加载**：使用 `loading="lazy"` 属性或 JavaScript 实现
- **深色模式**：使用 `prefers-color-scheme` 媒体查询或 JavaScript 切换
