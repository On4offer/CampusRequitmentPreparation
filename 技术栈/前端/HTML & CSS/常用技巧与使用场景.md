## 🌱 HTML 常用技巧

### 📖 语义化标签使用场景

**页面结构标签**：

| 标签 | 语义 | 使用场景 | 示例 |
|------|------|----------|------|
| `<header>` | 页面或区块头部 | 网站标题、导航栏 | 页面顶部、文章头部 |
| `<nav>` | 导航链接 | 主导航、面包屑 | 顶部导航栏、侧边菜单 |
| `<main>` | 主要内容 | 页面核心内容 | 文章内容、产品列表 |
| `<article>` | 独立内容 | 博客文章、新闻 | 单篇文章、评论 |
| `<section>` | 内容区块 | 章节、功能模块 | 产品特性、新闻列表 |
| `<aside>` | 侧边内容 | 相关链接、广告 | 侧边栏、推荐内容 |
| `<footer>` | 页面或区块底部 | 版权信息、联系方式 | 页面底部、文章结尾 |

**代码示例**：
```html
<!-- 完整页面结构 -->
<header>
    <h1>网站标题</h1>
    <nav>
        <a href="/">首页</a>
        <a href="/about">关于</a>
    </nav>
</header>

<main>
    <article>
        <h2>文章标题</h2>
        <section>
            <h3>章节标题</h3>
            <p>章节内容...</p>
        </section>
    </article>
    
    <aside>
        <h3>相关文章</h3>
        <ul>
            <li><a href="#">相关文章1</a></li>
        </ul>
    </aside>
</main>

<footer>
    <p>&copy; 2024 网站名称</p>
</footer>
```

**文本语义标签**：

| 标签 | 语义 | 使用场景 |
|------|------|----------|
| `<h1>-<h6>` | 标题层级 | 页面主标题到小节标题 |
| `<p>` | 段落 | 正文内容 |
| `<strong>` | 强调重要性 | 关键词、警告信息 |
| `<em>` | 强调语气 | 强调、引用 |
| `<mark>` | 标记文本 | 搜索结果、重点内容 |
| `<small>` | 小字体 | 版权、注释 |
| `<del>` | 删除文本 | 价格变动、已删除内容 |
| `<ins>` | 插入文本 | 新增内容 |
| `<code>` | 代码 | 内联代码 |
| `<pre>` | 预格式化 | 代码块、保留格式 |

### 📖 表单处理技巧

**表单基础结构**：
```html
<form action="/submit" method="POST">
    <div>
        <label for="username">用户名：</label>
        <input type="text" id="username" name="username" required>
    </div>
    <div>
        <label for="email">邮箱：</label>
        <input type="email" id="email" name="email" required>
    </div>
    <button type="submit">提交</button>
</form>
```

**输入类型与验证**：

| 输入类型 | 用途 | 验证特性 |
|----------|------|----------|
| `text` | 文本输入 | 基础文本 |
| `email` | 邮箱 | 邮箱格式验证 |
| `tel` | 电话 | 无内置验证 |
| `number` | 数字 | min、max、step |
| `date` | 日期 | 日期格式验证 |
| `password` | 密码 | 无内置验证 |
| `checkbox` | 复选框 | 可多个选中 |
| `radio` | 单选框 | 仅一个选中 |
| `file` | 文件上传 | accept、multiple |
| `search` | 搜索框 | 无特殊验证 |
| `url` | 网址 | URL格式验证 |

**验证属性**：
- `required`：必填
- `pattern`：正则表达式验证
- `min`/`max`：数值范围
- `minlength`/`maxlength`：字符长度
- `step`：步长（数字、日期）
- `placeholder`：占位符

**代码示例**：
```html
<!-- 带验证的表单 -->
<form>
    <!-- 用户名：3-20个字符 -->
    <div>
        <label for="username">用户名：</label>
        <input type="text" id="username" name="username" 
               required minlength="3" maxlength="20" 
               pattern="^[a-zA-Z0-9_]+$">
    </div>
    
    <!-- 密码：至少8位，包含字母和数字 -->
    <div>
        <label for="password">密码：</label>
        <input type="password" id="password" name="password" 
               required pattern="^(?=.*[A-Za-z])(?=.*\d)[A-Za-z\d]{8,}$">
    </div>
    
    <!-- 年龄：18-100岁 -->
    <div>
        <label for="age">年龄：</label>
        <input type="number" id="age" name="age" 
               required min="18" max="100" step="1">
    </div>
    
    <!-- 日期：过去的日期 -->
    <div>
        <label for="birthday">出生日期：</label>
        <input type="date" id="birthday" name="birthday" 
               max="2024-12-31">
    </div>
    
    <button type="submit">提交</button>
</form>
```

**文件上传**：
```html
<!-- 单文件上传 -->
<input type="file" name="avatar" accept="image/*">

<!-- 多文件上传 -->
<input type="file" name="files" multiple>

<!-- 限制文件类型 -->
<input type="file" name="document" accept=".pdf,.doc,.docx">
```

## 🌱 CSS 常用技巧

### 📖 布局技巧

**Flexbox 常用模式**：

| 布局需求 | 实现代码 | 适用场景 |
|----------|----------|----------|
| 水平居中 | `display: flex; justify-content: center;` | 导航菜单、按钮组 |
| 垂直居中 | `display: flex; align-items: center;` | 模态框、卡片内容 |
| 两端对齐 | `display: flex; justify-content: space-between;` | 导航栏、列表项 |
| 等宽布局 | `display: flex;` + `flex: 1;` | 均分容器空间 |
| 换行布局 | `display: flex; flex-wrap: wrap;` | 卡片网格、响应式列表 |

**代码示例**：
```css
/* 导航栏布局 */
.nav {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 0 20px;
    height: 60px;
}

.nav-links {
    display: flex;
    gap: 20px;
}

/* 卡片网格 */
.card-grid {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
}

.card {
    flex: 1 1 300px; /* 最小宽度300px */
    padding: 20px;
    border: 1px solid #ddd;
}
```

**Grid 常用模式**：

| 布局需求 | 实现代码 | 适用场景 |
|----------|----------|----------|
| 两栏布局 | `grid-template-columns: 200px 1fr;` | 侧边栏 + 内容 |
| 三栏布局 | `grid-template-columns: 200px 1fr 200px;` | 两侧边栏 + 内容 |
| 响应式网格 | `grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));` | 卡片、图片网格 |
| 固定表头 | `grid-template-rows: auto 1fr;` | 表格、仪表盘 |

**代码示例**：
```css
/* 页面布局 */
.page-layout {
    display: grid;
    grid-template-columns: 200px 1fr 250px;
    grid-template-rows: auto 1fr auto;
    grid-template-areas:
        "header header header"
        "nav main aside"
        "footer footer footer";
    min-height: 100vh;
}

.header { grid-area: header; }
.nav { grid-area: nav; }
.main { grid-area: main; }
.aside { grid-area: aside; }
.footer { grid-area: footer; }

/* 响应式卡片 */
.responsive-grid {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
    gap: 24px;
}
```

**定位技巧**：

| 定位类型 | 用途 | 实现代码 |
|----------|------|----------|
| 相对定位 | 微调位置 | `position: relative; top: 10px; left: 20px;` |
| 绝对定位 | 精确定位 | `position: absolute; top: 0; right: 0;` |
| 固定定位 | 固定显示 | `position: fixed; top: 0; left: 0; width: 100%;` |
| 粘性定位 | 滚动固定 | `position: sticky; top: 0;` |

**代码示例**：

```css
/* 固定导航栏 */
```

**选择器优化**：

| 选择器类型 | 性能 | 建议 |
|------------|------|------|
| ID选择器 | 高 | 用于唯一元素 |
| 类选择器 | 高 | 推荐使用 |
| 标签选择器 | 中 | 用于通用样式 |

| 子选择器 | 中 | 精准选择 |

**优化示例**：

```css
/* 不推荐：复杂选择器 */

    font-size: 1.2rem;

.post-lead {
    font-size: 1.2rem;
}
```

```css
}
```

### 📖 响应式单位

**相对单位**：

| 单位 | 参考对象 | 适用场景 |
|------|----------|----------|
| `rem` | 根元素字体大小 | 字体大小、间距 |
| `em` | 父元素字体大小 | 嵌套元素 |
| `vw` | 视口宽度 | 响应式宽度 |
| `vh` | 视口高度 | 响应式高度 |

**代码示例**：

```css
body {
    font-size: 1rem; /* 基础字体 */
}

h1 {
    font-size: 2rem; /* 2倍基础字体 */
}

/* 响应式容器 */
.hero {
    height: 80vh; /* 视口高度的80% */
    width: 100vw; /* 视口宽度的100% */
}

/* 响应式图片 */
.responsive-img {
    max-width: 100%;
    height: auto;
}
```

## 🌱 性能优化技巧

### 📖 CSS 性能优化

**代码优化**：
- **减少选择器复杂度**：避免过度嵌套
- **使用简写属性**：减少代码量
- **避免使用 @import**：会阻塞渲染
- **合并 CSS 文件**：减少 HTTP 请求
- **压缩 CSS**：减少文件大小

**渲染优化**：
- **避免重排**：使用 transform 代替 top/left
- **避免重绘**：减少使用触发重绘的属性
- **使用 will-change**：提示浏览器优化
- **使用硬件加速**：transform: translateZ(0)
- **使用 CSS containment**：隔离布局影响

**加载优化**：
```html
<!-- 预加载关键CSS -->
<link rel="preload" href="styles.css" as="style">

<!-- 异步加载CSS -->
<link rel="preload" href="non-critical.css" as="style"
      onload="this.onload=null;this.rel='stylesheet'">

<!-- 字体优化 -->
<link rel="preload" href="font.woff2" as="font" crossorigin>

<style>
@font-face {
    font-family: 'CustomFont';
    src: url('font.woff2') format('woff2');
    font-display: swap; /* 字体加载时显示系统字体 */
}
</style>
```

### 📖 图片优化

**图片格式选择**：
- **PNG**：适合图标、透明图片
- **JPEG**：适合照片、色彩丰富图片
- **WebP**：现代格式，体积更小
- **AVIF**：新一代格式，压缩率更高

**图片压缩**：
- 使用在线压缩工具
- 使用图片CDN服务
- 合理设置压缩质量

**懒加载**：
```html
<!-- 原生懒加载 -->
<img src="placeholder.jpg" loading="lazy" alt="描述文字">

<!-- JavaScript 懒加载 -->
<img data-src="image.jpg" class="lazyload" alt="描述文字">

<script>
// 简单的懒加载实现
const lazyImages = document.querySelectorAll('.lazyload');

if ('IntersectionObserver' in window) {
    const imageObserver = new IntersectionObserver((entries, observer) => {
        entries.forEach(entry => {
            if (entry.isIntersecting) {
                const img = entry.target;
                img.src = img.dataset.src;
                img.classList.remove('lazyload');
                imageObserver.unobserve(img);
            }
        });
    });
    
    lazyImages.forEach(img => imageObserver.observe(img));
}
</script>
```

```javascript
        if (e.key === 'Enter' || e.key === ' ') {
            e.preventDefault();
            this.click();
```

### 📖 表单处理解决方案

**1. 表单验证**：
```javascript
// 表单验证
const form = document.getElementById('contact-form');

form.addEventListener('submit', function(e) {
    e.preventDefault();
    
    const name = document.getElementById('name').value;
    const email = document.getElementById('email').value;
    const message = document.getElementById('message').value;
    
    let isValid = true;
    let errors = [];
    
    if (!name.trim()) {
        isValid = false;
        errors.push('请输入姓名');
    }
    
    if (!email.trim()) {
        isValid = false;
        errors.push('请输入邮箱');
    } else if (!/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email)) {
        isValid = false;
        errors.push('请输入有效的邮箱地址');
    }
    
    if (!message.trim()) {
        isValid = false;
        errors.push('请输入留言内容');
    }
    
    if (isValid) {
        // 提交表单
        alert('表单提交成功！');
        form.reset();
    } else {
        // 显示错误信息
        const errorContainer = document.getElementById('error-messages');
        errorContainer.innerHTML = errors.map(error => `<p>${error}</p>`).join('');
        errorContainer.style.display = 'block';
    }
});
```

**2. 表单提交**：
```javascript
// AJAX 表单提交
const form = document.getElementById('contact-form');

form.addEventListener('submit', function(e) {
    e.preventDefault();
    
    const formData = new FormData(form);
    const submitButton = form.querySelector('button[type="submit"]');
    
    // 显示加载状态
    submitButton.disabled = true;
    submitButton.textContent = '提交中...';
    
    fetch('/api/contact', {
        method: 'POST',
        body: formData
    })
    .then(response => response.json())
    .then(data => {
        if (data.success) {
            alert('表单提交成功！');
            form.reset();
        } else {
            alert('提交失败：' + data.message);
        }
    })
    .catch(error => {
        alert('提交失败：网络错误');
        console.error('Error:', error);
    })
    .finally(() => {
        // 恢复按钮状态
        submitButton.disabled = false;
        submitButton.textContent = '提交';
    });
});
```
