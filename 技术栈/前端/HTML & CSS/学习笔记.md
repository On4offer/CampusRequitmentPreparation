## 第1章：HTML 基础与语义化

### 1.1 HTML 简介

- **HTML（HyperText Markup Language）** 是用于创建网页结构的标记语言，由一系列标签组成，描述页面的结构和内容。
- **HTML5** 是当前主流版本，引入了语义化标签、Canvas、SVG、LocalStorage 等新特性，是前端开发的基础，也是校招面试的高频考点。
- **核心作用**：定义页面结构、提供内容语义、支持表单交互、嵌入媒体资源。

### 1.2 基本结构与文档规范

#### 标准文档结构
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <!-- 页面内容 -->
</body>
</html>
```

#### 文档类型与字符编码
- **DOCTYPE**：声明 HTML 版本，确保浏览器正确渲染
- **charset**：设置字符编码，推荐使用 UTF-8
- **viewport**：控制页面在移动设备上的显示

### 1.3 语义化标签

- **语义化的意义**：提高代码可读性、SEO 友好、无障碍访问
- **常用语义化标签**：
  - `<header>`：页面或区块的头部
  - `<nav>`：导航栏
  - `<main>`：主要内容区域
  - `<section>`：内容区块
  - `<article>`：独立的文章内容
  - `<aside>`：侧边栏或补充内容
  - `<footer>`：页面或区块的底部
  - `<figure>`：包含图片、图表等媒体内容
  - `<figcaption>`：媒体内容的说明文字

### 1.4 常用标签与使用场景

#### 文本标签
- `<h1>-<h6>`：标题，级别递减
- `<p>`：段落
- `<span>`：行内文本容器
- `<strong>`：强调（加粗）
- `<em>`：强调（斜体）
- `<br>`：换行
- `<hr>`：水平线

#### 链接与媒体
- `<a>`：链接，`href` 属性指定目标地址，`target` 属性控制打开方式
- `<img>`：图片，`src` 指向图片地址，`alt` 提供替代文本
- `<video>`：视频，支持 `controls`、`autoplay` 等属性
- `<audio>`：音频，支持 `controls`、`autoplay` 等属性

#### 列表
- `<ul>`：无序列表
- `<ol>`：有序列表
- `<li>`：列表项
- `<dl>`：定义列表
- `<dt>`：定义术语
- `<dd>`：定义描述

#### 表格
- `<table>`：表格容器
- `<thead>`：表头
- `<tbody>`：表格主体
- `<tr>`：表格行
- `<th>`：表头单元格
- `<td>`：数据单元格

#### 表单
- `<form>`：表单容器，`action` 指定提交地址，`method` 指定提交方式
- `<input>`：输入框，`type` 属性控制类型（text、password、email、number 等）
- `<textarea>`：多行文本输入
- `<select>`：下拉选择框
- `<option>`：选择项
- `<button>`：按钮
- `<label>`：表单标签，与输入控件关联

### 1.5 全局属性与数据属性

- **全局属性**：所有 HTML 元素都支持的属性
  - `id`：唯一标识符
  - `class`：类名，用于样式和选择器
  - `style`：内联样式
  - `title`：鼠标悬停时的提示文本
  - `data-*`：自定义数据属性，用于存储页面或应用的私有数据
  - `aria-*`：无障碍属性，提高可访问性

- **数据属性使用示例**：
```html
<div data-user-id="123" data-user-name="张三">用户信息</div>
<script>
    const div = document.querySelector('div');
    console.log(div.dataset.userId); // 输出：123
    console.log(div.dataset.userName); // 输出：张三
</script>
```

## 第2章：CSS 基础与选择器

### 2.1 CSS 简介

- **CSS（Cascading Style Sheets）** 用于描述 HTML 元素的样式，包括布局、颜色、字体等。
- **CSS3** 引入了圆角、阴影、动画、渐变等新特性，增强了样式能力，是现代前端开发的核心技术之一。
- **CSS 工作原理**：浏览器解析 HTML 生成 DOM 树，解析 CSS 生成 CSSOM 树，两者结合生成渲染树，最终绘制页面。

### 2.2 CSS 引入方式

1. **内联样式**：直接在 HTML 元素的 `style` 属性中定义
   ```html
   <div style="color: red; font-size: 16px;">内联样式</div>
   ```

2. **内部样式**：在 HTML 的 `<head>` 中使用 `<style>` 标签定义
   ```html
   <style>
       div {
           color: red;
           font-size: 16px;
       }
   </style>
   ```

3. **外部样式**：使用 `<link>` 标签引入外部 CSS 文件
   ```html
   <link rel="stylesheet" href="style.css">
   ```

4. **@import**：在 CSS 文件中引入其他 CSS 文件
   ```css
   @import url('other.css');
   ```

### 2.3 选择器

#### 基础选择器
1. **元素选择器**：根据元素标签名选择
   ```css
   p { color: blue; }
   ```

2. **ID 选择器**：根据元素的 `id` 属性选择
   ```css
   #header { background: #f0f0f0; }
   ```

3. **类选择器**：根据元素的 `class` 属性选择
   ```css
   .container { width: 100%; }
   ```

4. **通配符选择器**：选择所有元素
   ```css
   * { margin: 0; padding: 0; }
   ```

#### 组合选择器
1. **后代选择器**：选择所有后代元素
   ```css
   .container p { color: gray; }
   ```

2. **子选择器**：只选择直接子元素
   ```css
   .container > p { color: gray; }
   ```

3. **相邻兄弟选择器**：选择相邻的下一个兄弟元素
   ```css
   h1 + p { margin-top: 10px; }
   ```

4. **通用兄弟选择器**：选择所有后续兄弟元素
   ```css
   h1 ~ p { color: gray; }
   ```

#### 属性选择器
1. **存在属性**：选择具有指定属性的元素
   ```css
   [title] { cursor: help; }
   ```

2. **属性值等于**：选择属性值等于指定值的元素
   ```css
   [type="text"] { width: 200px; }
   ```

3. **属性值开头**：选择属性值以指定值开头的元素
   ```css
   [href^="https"] { color: green; }
   ```

4. **属性值结尾**：选择属性值以指定值结尾的元素
   ```css
   [href$=".pdf"] { color: red; }
   ```

5. **属性值包含**：选择属性值包含指定值的元素
   ```css
   [class*="btn"] { padding: 5px; }
   ```

#### 伪类选择器
1. **状态伪类**：
   ```css
   :hover { color: red; } /* 鼠标悬停 */
   :active { color: blue; } /* 激活状态 */
   :focus { outline: 2px solid green; } /* 获得焦点 */
   ```

2. **结构伪类**：
   ```css
   :first-child { margin-top: 0; } /* 第一个子元素 */
   :last-child { margin-bottom: 0; } /* 最后一个子元素 */
   :nth-child(2n) { background: #f0f0f0; } /* 偶数子元素 */
   :nth-child(2n+1) { background: #fff; } /* 奇数子元素 */
   ```

3. **否定伪类**：
   ```css
   :not(.active) { opacity: 0.5; } /* 不包含 active 类的元素 */
   ```

#### 伪元素选择器
1. **内容伪元素**：
   ```css
   ::before { content: "前缀"; }
   ::after { content: "后缀"; }
   ```

2. **文本伪元素**：
   ```css
   ::first-line { font-weight: bold; } /* 第一行 */
   ::first-letter { font-size: 2em; } /* 第一个字母 */
   ```

### 2.4 样式优先级与层叠规则

#### 优先级计算
- **优先级权重**：
  - `!important`：最高优先级
  - 内联样式：1000
  - ID 选择器：100
  - 类选择器、属性选择器、伪类：10
  - 元素选择器、伪元素：1
  - 通配符：0

- **计算示例**：
  - `#header .nav li`：100 + 10 + 1 = 111
  - `.container p:first-child`：10 + 1 + 10 = 21

#### 层叠规则
1. **优先级高的样式覆盖优先级低的**
2. **同优先级时，后定义的覆盖先定义的**
3. **继承的样式优先级低于直接定义的**

#### 继承
- **可继承属性**：字体相关（font-*）、颜色（color）、文本相关（text-*）等
- **不可继承属性**：布局相关（width、height、margin、padding等）、边框（border）等
- **强制继承**：使用 `inherit` 关键字
  ```css
  .child { color: inherit; } /* 继承父元素的颜色 */
  ```

## 第3章：CSS 盒模型与布局

### 3.1 盒模型

#### 盒模型概念
- **盒模型**：CSS 中每个元素都被视为一个盒子，由内容区、内边距、边框和外边距组成
- **标准盒模型**：`width` 和 `height` 仅指内容区的宽度和高度
- **IE 盒模型**：`width` 和 `height` 包含内容区、内边距和边框

#### box-sizing 属性
- **content-box**：标准盒模型（默认）
- **border-box**：IE 盒模型，推荐使用

```css
/* 推荐全局设置 */
* {
    box-sizing: border-box;
}
```

#### 盒模型组成
1. **内容区（content）**：显示内容的区域，由 `width` 和 `height` 控制
2. **内边距（padding）**：内容区与边框之间的空间
3. **边框（border）**：围绕内容区和内边距的边界
4. **外边距（margin）**：盒子与其他元素之间的空间

#### 盒模型计算
- **总宽度**：`width + padding-left + padding-right + border-left + border-right`
- **总高度**：`height + padding-top + padding-bottom + border-top + border-bottom`

### 3.2 传统布局技术

#### 浮动布局（float）
- **作用**：使元素向左或向右浮动，脱离文档流
- **常用属性**：`float: left | right | none`
- **清除浮动**：
  ```css
  /* 方法1：clear 属性 */
  .clear { clear: both; }
  
  /* 方法2：伪元素清除 */
  .container::after {
      content: "";
      display: block;
      clear: both;
  }
  
  /* 方法3：overflow 属性 */
  .container { overflow: hidden; }
  ```

#### 定位布局（position）
- **static**：默认值，正常文档流
- **relative**：相对定位，相对于自身原来的位置
- **absolute**：绝对定位，相对于最近的非 static 定位祖先元素
- **fixed**：固定定位，相对于浏览器窗口
- **sticky**：粘性定位，结合了 relative 和 fixed 的特性

#### 显示属性（display）
- **block**：块级元素，独占一行
- **inline**：行内元素，不独占一行
- **inline-block**：行内块元素，兼具行内和块级特性
- **none**：隐藏元素
- **flex**：弹性布局
- **grid**：网格布局

### 3.3 Flexbox 弹性布局

#### Flexbox 概念
- **Flexbox**：一维布局模型，用于灵活排列元素
- **容器与项目**：设置 `display: flex` 的元素为容器，其直接子元素为项目

#### 容器属性
1. **flex-direction**：主轴方向
   ```css
   flex-direction: row | row-reverse | column | column-reverse
   ```

2. **justify-content**：主轴对齐方式
   ```css
   justify-content: flex-start | flex-end | center | space-between | space-around
   ```

3. **align-items**：交叉轴对齐方式
   ```css
   align-items: flex-start | flex-end | center | baseline | stretch
   ```

4. **flex-wrap**：是否换行
   ```css
   flex-wrap: nowrap | wrap | wrap-reverse
   ```

5. **align-content**：多根轴线的对齐方式
   ```css
   align-content: flex-start | flex-end | center | space-between | space-around | stretch
   ```

#### 项目属性
1. **flex-grow**：项目的放大比例
2. **flex-shrink**：项目的缩小比例
3. **flex-basis**：项目的初始宽度
4. **flex**：`flex-grow`、`flex-shrink`、`flex-basis` 的简写
5. **align-self**：单个项目的交叉轴对齐方式

#### 常用布局示例
- **水平居中**：`justify-content: center`
- **垂直居中**：`align-items: center`
- **两端对齐**：`justify-content: space-between`
- **等宽布局**：`flex: 1`

### 3.4 Grid 网格布局

#### Grid 概念
- **Grid**：二维布局模型，使用行和列来排列元素
- **容器与项目**：设置 `display: grid` 的元素为容器，其直接子元素为项目

#### 容器属性
1. **grid-template-columns**：定义列的宽度
   ```css
   grid-template-columns: 100px 1fr 2fr
   ```

2. **grid-template-rows**：定义行的高度
   ```css
   grid-template-rows: 50px 100px
   ```

3. **grid-gap**：行列间距（简写）
   ```css
   grid-gap: 10px
   ```

4. **grid-template-areas**：定义网格区域
   ```css
   grid-template-areas: 
       "header header header"
       "sidebar content content"
       "footer footer footer"
   ```

5. **justify-items**：单元格内容的水平对齐方式
6. **align-items**：单元格内容的垂直对齐方式
7. **justify-content**：整个网格的水平对齐方式
8. **align-content**：整个网格的垂直对齐方式

#### 项目属性
1. **grid-column-start/end**：项目的列起始和结束位置
2. **grid-row-start/end**：项目的行起始和结束位置
3. **grid-column**：`grid-column-start / grid-column-end` 的简写
4. **grid-row**：`grid-row-start / grid-row-end` 的简写
5. **grid-area**：指定项目所在的网格区域

#### 常用布局示例
- **两栏布局**：
  ```css
  .container {
      display: grid;
      grid-template-columns: 200px 1fr;
  }
  ```

- **三栏布局**：
  ```css
  .container {
      display: grid;
      grid-template-columns: 200px 1fr 200px;
  }
  ```

- **响应式布局**：
  ```css
  .container {
      display: grid;
      grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  }
  ```

## 第4章：CSS 高级特性

### 4.1 动画与过渡

#### 过渡（transition）
- **作用**：使元素属性变化更加平滑
- **属性**：
  - `transition-property`：要过渡的属性
  - `transition-duration`：过渡持续时间
  - `transition-timing-function`：过渡 timing 函数
  - `transition-delay`：过渡延迟时间

- **简写**：
  ```css
  transition: property duration timing-function delay;
  ```

- **示例**：
  ```css
  .btn {
      transition: all 0.3s ease;
  }
  
  .btn:hover {
      transform: scale(1.1);
      background-color: #3498db;
  }
  ```

#### 动画（animation）
- **作用**：创建复杂的动画效果
- **步骤**：
  1. 定义动画关键帧
  2. 应用动画到元素

- **定义关键帧**：
  ```css
  @keyframes fadeIn {
      from { opacity: 0; }
      to { opacity: 1; }
  }
  ```

- **应用动画**：
  ```css
  .element {
      animation: fadeIn 1s ease-in-out;
  }
  ```

- **动画属性**：
  - `animation-name`：动画名称
  - `animation-duration`：动画持续时间
  - `animation-timing-function`：动画 timing 函数
  - `animation-delay`：动画延迟时间
  - `animation-iteration-count`：动画重复次数
  - `animation-direction`：动画方向
  - `animation-fill-mode`：动画结束后的状态
  - `animation-play-state`：动画播放状态

- **简写**：
  ```css
  animation: name duration timing-function delay iteration-count direction fill-mode;
  ```

### 4.2 渐变与阴影

#### 渐变（gradient）
- **线性渐变**：
  ```css
  background: linear-gradient(to right, red, blue);
  background: linear-gradient(45deg, red, blue);
  ```

- **径向渐变**：
  ```css
  background: radial-gradient(circle, red, blue);
  background: radial-gradient(ellipse at center, red, blue);
  ```

- **重复渐变**：
  ```css
  background: repeating-linear-gradient(45deg, red, blue 20px);
  background: repeating-radial-gradient(circle, red, blue 20px);
  ```

#### 阴影（shadow）
- **文本阴影**：
  ```css
  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
  ```

- **盒阴影**：
  ```css
  box-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
  box-shadow: inset 2px 2px 4px rgba(0, 0, 0, 0.5); /* 内阴影 */
  ```

### 4.3 变换（transform）

#### 2D 变换
- **平移**：
  ```css
  transform: translate(10px, 20px);
  ```

- **旋转**：
  ```css
  transform: rotate(45deg);
  ```

- **缩放**：
  ```css
  transform: scale(1.1);
  ```

- **倾斜**：
  ```css
  transform: skew(10deg, 5deg);
  ```

- **组合变换**：
  ```css
  transform: translate(10px, 20px) rotate(45deg) scale(1.1);
  ```

#### 3D 变换
- **透视**：
  ```css
  perspective: 1000px;
  ```

- **3D 旋转**：
  ```css
  transform: rotateX(45deg);
  transform: rotateY(45deg);
  transform: rotateZ(45deg);
  ```

- **3D 平移**：
  ```css
  transform: translate3d(10px, 20px, 30px);
  ```

- **3D 缩放**：
  ```css
  transform: scale3d(1.1, 1.1, 1.1);
  ```

### 4.4 媒体查询（media query）

#### 媒体查询概念
- **媒体查询**：根据设备特性（如屏幕宽度、高度、方向等）应用不同的样式
- **作用**：实现响应式设计

#### 基本语法
```css
@media media-type and (media-feature) {
    /* 样式规则 */
}
```

#### 常用媒体特性
- **width/height**：视口宽度/高度
- **min-width/max-width**：最小/最大视口宽度
- **min-height/max-height**：最小/最大视口高度
- **orientation**：设备方向（portrait/landscape）
- **aspect-ratio**：宽高比
- **resolution**：设备分辨率

#### 响应式设计示例
```css
/* 移动设备 */
@media (max-width: 767px) {
    .container {
        width: 100%;
    }
}

/* 平板设备 */
@media (min-width: 768px) and (max-width: 991px) {
    .container {
        width: 750px;
    }
}

/* 桌面设备 */
@media (min-width: 992px) {
    .container {
        width: 970px;
    }
}

/* 大屏幕设备 */
@media (min-width: 1200px) {
    .container {
        width: 1170px;
    }
}
```

### 4.5 CSS 变量

#### CSS 变量概念
- **CSS 变量**：使用 `--` 前缀定义的变量，可在整个文档中重复使用
- **作用**：提高代码可维护性，方便主题切换

#### 定义与使用
- **定义变量**：
  ```css
  :root {
      --primary-color: #3498db;
      --secondary-color: #2ecc71;
      --font-size: 16px;
  }
  ```

- **使用变量**：
  ```css
  .btn {
      background-color: var(--primary-color);
      color: white;
      font-size: var(--font-size);
  }
  ```

- **变量作用域**：
  - 全局作用域：在 `:root` 中定义
  - 局部作用域：在特定选择器中定义

- **变量计算**：
  ```css
  .element {
      width: calc(var(--base-width) * 2);
      height: calc(var(--base-height) + 10px);
  }
  ```

## 第5章：HTML & CSS 实战技巧

### 5.1 常见布局解决方案

#### 居中布局
- **水平居中**：
  ```css
  /* 行内元素 */
  .parent {
      text-align: center;
  }
  
  /* 块级元素 */
  .child {
      margin: 0 auto;
      width: 200px;
  }
  
  /* Flexbox */
  .parent {
      display: flex;
      justify-content: center;
  }
  ```

- **垂直居中**：
  ```css
  /* 单行文本 */
  .parent {
      height: 100px;
      line-height: 100px;
  }
  
  /* Flexbox */
  .parent {
      display: flex;
      align-items: center;
  }
  
  /* Grid */
  .parent {
      display: grid;
      place-items: center;
  }
  
  /* 绝对定位 */
  .parent {
      position: relative;
      height: 200px;
  }
  
  .child {
      position: absolute;
      top: 50%;
      transform: translateY(-50%);
  }
  ```

- **水平垂直居中**：
  ```css
  /* Flexbox */
  .parent {
      display: flex;
      justify-content: center;
      align-items: center;
      height: 200px;
  }
  
  /* Grid */
  .parent {
      display: grid;
      place-items: center;
      height: 200px;
  }
  
  /* 绝对定位 */
  .parent {
      position: relative;
      height: 200px;
  }
  
  .child {
      position: absolute;
      top: 50%;
      left: 50%;
      transform: translate(-50%, -50%);
  }
  ```

#### 响应式布局
- **媒体查询**：根据屏幕尺寸调整样式
- **Flexbox**：灵活的一维布局
- **Grid**：强大的二维布局
- **REM 单位**：相对于根元素字体大小的单位
- **VW/VH**：相对于视口宽度/高度的单位

#### 多列布局
- **浮动布局**：
  ```css
  .column {
      float: left;
      width: 33.33%;
      padding: 10px;
  }
  ```

- **Flexbox**：
  ```css
  .container {
      display: flex;
  }
  
  .column {
      flex: 1;
      padding: 10px;
  }
  ```

- **Grid**：
  ```css
  .container {
      display: grid;
      grid-template-columns: repeat(3, 1fr);
      gap: 10px;
  }
  ```

### 5.2 性能优化

#### CSS 性能优化
- **减少选择器复杂度**：避免过度嵌套
- **使用 CSS 变量**：提高代码可维护性
- **避免使用 @import**：会阻塞页面渲染
- **使用简写属性**：减少代码量
- **压缩 CSS**：减少文件大小

#### 渲染性能优化
- **避免重排（reflow）**：
  - 避免频繁操作 DOM
  - 使用文档片段（DocumentFragment）
  - 避免频繁读取布局属性
  - 使用 CSS transform 代替 top/left 等属性

- **避免重绘（repaint）**：
  - 减少使用触发重绘的属性（如 color、background-color）
  - 使用 will-change 属性提示浏览器

#### 加载性能优化
- **内联关键 CSS**：首屏关键样式内联到 HTML 中
- **使用 CSS 预处理器**：如 Sass、Less
- **使用 CSS Modules**：避免样式冲突
- **使用 CSS-in-JS**：组件化样式管理

### 5.3 浏览器兼容性

#### 浏览器前缀
- **常用前缀**：-webkit-、-moz-、-ms-、-o-
- **使用 Autoprefixer**：自动添加浏览器前缀

#### 兼容性解决方案
- **使用 Modernizr**：检测浏览器特性
- **使用 Polyfill**：为旧浏览器提供新特性支持
- **优雅降级**：为不支持的浏览器提供替代方案

### 5.4 无障碍访问

#### 语义化 HTML
- 使用正确的语义化标签
- 为图片添加 alt 属性
- 为表单元素添加 label

#### ARIA 属性
- **ARIA 角色**：`role="navigation"`、`role="button"` 等
- **ARIA 状态**：`aria-hidden="true"`、`aria-expanded="false"` 等
- **ARIA 属性**：`aria-label`、`aria-labelledby` 等

#### 键盘可访问性
- 确保所有交互元素可通过键盘访问
- 提供清晰的焦点状态
- 支持键盘导航
