# 📘《计算机网络与浏览器学习与实战手册》

> 系统学习见下文各章；日常常用概念与面试要点速查见同目录《常用概念与使用场景》。

------

## 第1章：计算机网络基础

### 1.1 分层模型

- **OSI 七层**：物理层、数据链路层、网络层、传输层、会话层、表示层、应用层。
- **TCP/IP 四层**：网络接口层、网际层（IP）、传输层（TCP/UDP）、应用层（HTTP、DNS 等）。前端开发与校招常关注**应用层与传输层**。

### 1.2 IP、端口与 DNS

- **IP**：标识主机；**端口**：标识进程；**DNS**：域名 → IP，浏览器缓存、本地 hosts、递归/迭代查询。

## 第2章：HTTP 协议

1. 请求与响应：请求方法（GET、POST、PUT、DELETE 等）、状态码（2xx、3xx、4xx、5xx）、请求/响应头
2. 常用头：Content-Type、Accept、Cache-Control、Cookie、Set-Cookie、CORS 相关（Origin、Access-Control-Allow-*）
3. HTTP/1.1 与 HTTP/2：持久连接、多路复用、头部压缩
4. HTTPS：TLS/SSL 加密、握手过程简述、证书与 CA

------

## 第3章：缓存机制

1. 强缓存：Expires、Cache-Control（max-age、no-cache、no-store、private/public）
2. 协商缓存：Last-Modified / If-Modified-Since、ETag / If-None-Match；304 与 200
3. 缓存策略：前端资源 hash、接口缓存与不缓存场景、Service Worker 简述
4. 面试要点：强缓存与协商缓存流程、如何禁用缓存、CDN 与缓存

------

## 第4章：跨域与安全

1. 同源策略：协议、域名、端口一致才同源；限制：Cookie/LocalStorage、DOM、Ajax
2. 跨域方案：CORS（预检请求、响应头）、JSONP、代理（开发 proxy、Nginx 反向代理）、postMessage
3. 安全：XSS（转义、CSP、HttpOnly）、CSRF（Token、SameSite Cookie）、HTTPS

------

## 第5章：浏览器渲染与性能

1. 渲染流程：解析 HTML → DOM、CSS → CSSOM、合成渲染树 → 布局 → 绘制 → 合成
2. 重排（reflow）与重绘（repaint）：触发条件、如何减少（批量读写、脱离文档流、will-change）
3. 首屏与性能指标：白屏时间、FCP、LCP、FID/INP、CLS；优化思路：资源加载、懒加载、代码分割
4. 事件循环（与 JS 结合）：宏任务、微任务、渲染时机

------

## 第6章：浏览器存储与其它

1. Cookie：大小约 4KB、可设过期与 HttpOnly/Secure/SameSite、同源与跨域携带
2. LocalStorage / SessionStorage：容量约 5MB、同源、SessionStorage 标签页级
3. 跨标签页通信：LocalStorage 事件、BroadcastChannel、SharedWorker、postMessage
4. 浏览器兼容与调试：DevTools 网络/性能、兼容性查询（caniuse）

------

## 第7章：实战与面试要点

1. 从输入 URL 到页面展示：DNS、TCP 连接、HTTP 请求、响应、解析渲染、连接关闭
2. 常见状态码含义与使用场景（301/302、304、401、403、404、500）
3. 面试高频：强缓存与协商缓存、CORS 预检、XSS 与 CSRF 防护、重排重绘与优化
