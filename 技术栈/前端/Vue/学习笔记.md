## 第1章：Vue 概述与核心思想

### 1.1 Vue 简介

- **Vue** 是一套用于构建用户界面的**渐进式前端框架**，由尤雨溪创建，核心库只关注视图层，易于与现有项目或其它库集成。
- **Vue 3** 为主流版本：Composition API、更好的 TypeScript 支持、性能与 Tree-shaking 优化。

### 1.2 核心特点

- **响应式**：数据驱动视图，数据变更自动更新 DOM；**组件化**：页面由组件拼装，可复用与组合；**声明式**：模板 + 数据绑定，减少手写 DOM；**渐进式**：可从局部使用到整站框架。

### 1.3 与 React、Angular 简要对比

- **Vue**：模板语法友好、上手快、中文生态好；**React**：JSX、生态大、灵活；**Angular**：全家桶、企业级。Vue 适合中后台、移动端 H5、与 Java 后端配合的前端项目。
