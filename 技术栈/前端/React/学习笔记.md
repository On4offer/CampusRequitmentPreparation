## 第1章：React 概述与核心思想

### 1.1 React 简介

- **React** 是 Meta（Facebook）推出的**用于构建用户界面的 JavaScript 库**，采用组件化、声明式与“数据驱动视图”的思想，生态庞大，在 AI 应用、中后台、大厂前端中广泛使用。
- **React 18** 为主流：函数组件 + Hooks 为默认写法、Concurrent 特性、自动批处理等。

### 1.2 核心特点

- **组件化**：界面由组件组合而成，可复用与拆分；**声明式**：描述“UI 在数据下的样子”，由 React 负责更新 DOM；**单向数据流**：数据自顶向下，通过 props 传递，变更通过 setState/更新 state 触发重渲染；**Learn Once, Write Anywhere**：可配合 React Native 做移动端、与 Node 做 SSR。

### 1.3 与 Vue 简要对比

- **React**：JSX、Hooks、生态与 AI 工具链（如 Vercel AI SDK）更丰富，招聘与开源示例多；**Vue**：模板语法友好、上手快。两者均能胜任中后台与 AI 应用前端，按团队与生态选型即可。

## 第3章：Hooks 与状态

1. useState：状态声明与更新、批量更新、不可变更新
2. useEffect：副作用、依赖数组、清理函数（return）
3. useContext：跨层级消费 Context
4. useRef：DOM 引用与持久化值（不触发重渲染）
5. useMemo / useCallback：性能优化与稳定引用
