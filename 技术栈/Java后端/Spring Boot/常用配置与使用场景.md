# 📌 Spring Boot 常用配置与使用场景速查

> 配合《学习笔记》使用，侧重**日常开发中的配置项、注解与场景**，便于速查与上手。

## 一、常用配置项（application.yml / properties）

### 1.1 服务与端口

```yaml
server:
  port: 8080
  servlet:
    context-path: /api    # 上下文路径，可选
```

### 1.2 数据源（HikariCP）

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/db?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai
    username: root
    password: xxx
    driver-class-name: com.mysql.cj.jdbc.Driver
    hikari:
      minimum-idle: 5
      maximum-pool-size: 20
      connection-timeout: 30000
```

### 1.3 Redis

```yaml
spring:
  redis:
    host: localhost
    port: 6379
    password: 
    database: 0
    lettuce:
      pool:
        max-active: 8
        max-idle: 8
        min-idle: 0
```

### 1.4 多环境 Profile

```yaml
# 激活 profile
spring:
  profiles:
    active: dev   # 或通过命令行 --spring.profiles.active=prod
```

- 文件命名：**application-dev.yml**、**application-prod.yml**；激活 dev 时加载 application.yml + application-dev.yml。

### 1.5 日志

```yaml
logging:
  level:
    root: INFO
    com.example: DEBUG
    org.springframework.web: INFO
  file:
    name: logs/app.log
  pattern:
    console: "%d{yyyy-MM-dd HH:mm:ss} [%thread] %-5level %logger{36} - %msg%n"
```

### 1.6 Actuator（监控）

```yaml
management:
  endpoints:
    web:
      exposure:
        include: health,info   # 生产建议只开 health、info
  endpoint:
    health:
      show-details: when_authorized  # 或 always / never
```

## 二、常用注解速查

### 2.1 启动与配置类

| 注解 | 说明 | 使用场景 |
|------|------|----------|
| **@SpringBootApplication** | 启动类，组合 @Configuration + @EnableAutoConfiguration + @ComponentScan | 主类 |
| **@Configuration** | 配置类，内部可写 @Bean | Java 配置 |
| **@ConfigurationProperties(prefix="app")** | 将配置前缀绑定到 Bean 属性 | 结构化配置（如 app.redis.*） |
| **@EnableConfigurationProperties(X.class)** | 启用某配置属性类并注册为 Bean | 配合上面使用 |

### 2.2 配置注入

| 注解 | 说明 | 使用场景 |
|------|------|----------|
| **@Value("${key}")** | 注入单个配置项 | 简单值、开关 |
| **@Value("${key:default}")** | 带默认值 | 可选配置 |

### 2.3 条件与排除

| 注解 | 说明 | 使用场景 |
|------|------|----------|
| **@ConditionalOnProperty("key")** | 配置存在且匹配时生效 | 功能开关 |
| **@ConditionalOnClass(X.class)** | classpath 有该类时生效 | Starter 中按依赖装配 |
| **@Profile("dev")** | 仅指定 profile 激活时注册 Bean | 多环境不同实现 |
| **exclude = XxxAutoConfiguration.class** | @SpringBootApplication 排除自动配置 | 解决冲突或关闭某功能 |

## 三、常用 Starter 依赖（Maven）

| 依赖 | 说明 | 使用场景 |
|------|------|----------|
| **spring-boot-starter-web** | Web + Tomcat + Jackson | REST 接口 |
| **spring-boot-starter-data-jpa** | JPA + Hibernate | 关系型 ORM |
| **spring-boot-starter-data-redis** | Redis + Lettuce | 缓存、分布式锁 |
| **spring-boot-starter-validation** | JSR-303 校验 | @Valid、@NotNull 等 |
| **spring-boot-starter-actuator** | 监控端点 | 健康检查、指标 |
| **spring-boot-starter-security** | Spring Security | 认证授权 |
| **mybatis-spring-boot-starter** | MyBatis 集成 | SQL 映射 |
| **spring-boot-starter-cache** | 抽象缓存 | @Cacheable 等 |

## 四、使用场景对照

| 场景 | 推荐做法 |
|------|----------|
| 多环境切换 | application-{profile}.yml + spring.profiles.active，或 -Dspring.profiles.active=prod |
| 配置敏感信息 | 环境变量、配置中心，或加密后注入（如 Jasypt、Spring Cloud Config） |
| 自定义配置绑定 | @ConfigurationProperties(prefix="app.xxx") + @EnableConfigurationProperties |
| 关闭某自动配置 | @SpringBootApplication(exclude = XxxAutoConfiguration.class) |
| 健康检查 | 引入 actuator，暴露 /actuator/health；K8s 用做 liveness/readiness |
| 自定义指标 | 注入 MeterRegistry，注册 Counter/Timer/Gauge；配合 Prometheus 拉取 |
| 全局异常与统一返回 | @RestControllerAdvice + @ExceptionHandler，返回统一 Result |
| 拦截器 / 过滤器 | WebMvcConfigurer.addInterceptors；Filter 注册 FilterRegistrationBean |
| 跨域 | WebMvcConfigurer.addCorsMappings 或 @CrossOrigin |
| 文件上传大小 | spring.servlet.multipart.max-file-size、max-request-size |

## 五、常用命令与参数

| 用途 | 命令或参数 |
|------|------------|
| 指定 Profile | `--spring.profiles.active=prod` 或 `-Dspring.profiles.active=prod` |
| 指定端口 | `--server.port=9090` |
| 开启自动配置报告 | `--debug` 启动，查看 ConditionEvaluationReport |
| 懒加载（全局） | `spring.main.lazy-initialization=true`（慎用） |
| 运行 Jar | `java -jar app.jar` 或 `java -Xmx512m -jar app.jar` |
