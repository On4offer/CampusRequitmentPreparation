## 🧩 一、题意理解

> 按照从上到下、从左到右的顺序，依次遍历二叉树每一层节点，返回每层节点值的集合。

------

### 示例：

输入：

```
     3
   /   \
  9     20
       /  \
      15   7
```

输出：

```
[
  [3],
  [9,20],
  [15,7]
]
```

------

## 💡 二、思路分析 — 层序遍历（BFS）

层序遍历即：

> 从根节点开始，一层一层遍历二叉树的节点。

要想“一层一层”，就需要借助 **队列（Queue）** 实现 **广度优先搜索 BFS**。

------

### 🔁 核心步骤

1. 创建结果列表 `List<List<Integer>> res`
2. 若根节点为空，直接返回空列表
3. 创建队列 `Queue<TreeNode>`，把根节点入队
4. 当队列非空：
   - 取出当前层的节点数量 `size`
   - 循环 `size` 次，依次出队节点并将值加入当前层 `list`
   - 将出队节点的左右子节点入队
   - 当前层遍历完后，将 `list` 加入结果 `res`

------

## 🧩 三、伪代码

```
if root == null: return []

queue = [root]
res = []

while queue not empty:
    level = []
    size = queue.size()
    for i in 1..size:
        node = queue.poll()
        level.add(node.val)
        if node.left != null: queue.offer(node.left)
        if node.right != null: queue.offer(node.right)
    res.add(level)

return res
```

------

## ✅ 四、Java 实现（标准 BFS）

```java
import java.util.*;

class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        if (root == null) return res;

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);

        while (!queue.isEmpty()) {
            int size = queue.size();              // 当前层节点个数
            List<Integer> level = new ArrayList<>(); // 存储当前层节点值

            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();     // 出队当前节点
                level.add(node.val);              // 记录节点值

                if (node.left != null) queue.offer(node.left);   // 左子节点入队
                if (node.right != null) queue.offer(node.right); // 右子节点入队
            }

            res.add(level); // 当前层遍历完毕，加入结果
        }

        return res;
    }
}
```

## 🔄 五、执行流程演示

以树：

```
     3
   /   \
  9     20
       /  \
      15   7
```

执行步骤如下：

| 阶段   | 队列内容                | 当前层结果 | 总结果              |
| ------ | ----------------------- | ---------- | ------------------- |
| 初始化 | [3]                     | -          | []                  |
| 第1层  | 出队3 → 入队9,20        | [3]        | [[3]]               |
| 第2层  | 出队9,20 → 入队15,7     | [9,20]     | [[3],[9,20]]        |
| 第3层  | 出队15,7 → 无子节点入队 | [15,7]     | [[3],[9,20],[15,7]] |

最终返回：

```
[[3],[9,20],[15,7]]
```

| 项目       | 分析                           |
| ---------- | ------------------------------ |
| 时间复杂度 | O(n) —— 每个节点入队出队各一次 |
| 空间复杂度 | O(n) —— 队列最多存储一层节点   |

## 💬 七、面试回答模板

> 这题考察的是二叉树的 **广度优先遍历（BFS）**。
>
> 思路是使用队列，按层次依次取出节点并记录每层的节点值。
>  每一轮循环代表一层节点的处理，时间复杂度为 O(n)，空间复杂度 O(n)。

------

## 🔍 八、面试常见追问

| 追问                       | 答法                                                         |
| -------------------------- | ------------------------------------------------------------ |
| 🔹 如果要“自底向上”遍历？   | 可以在最后 `Collections.reverse(res)` 或用 `LinkedList` 每层 `addFirst()` |
| 🔹 如果要“Z 字形层序遍历”？ | 每层记录层号，偶数层正序加入，奇数层反序加入                 |
| 🔹 如果要求统计每层平均值？ | 用 `sum/size` 计算平均值即可                                 |
