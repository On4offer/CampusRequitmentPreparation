 题目是：**两两交换链表中的节点（Swap Nodes in Pairs）**

------

## 💡题意简述

给一个单链表，每两个相邻节点交换位置，返回新的头节点。
 要求：**不能修改节点值**，只能改指针。

------

## 🧠思路（dummy + 三指针法）

链表题的套路：

> 凡是可能要“动头节点”的，都加一个虚拟头节点 dummy。

我们用三个指针来操作：

- `prev`：当前这对节点的前驱（初始是 dummy）
- `first`：当前要交换的第一个节点
- `second`：当前要交换的第二个节点

目标是：

```
prev -> first -> second -> next
```

变成

```
prev -> second -> first -> next
```

------

## ✅完整实现代码

```java
class Solution {
    public ListNode swapPairs(ListNode head) {
        // 创建虚拟头节点，方便处理头节点被交换的情况
        ListNode dummy = new ListNode(0, head);
        // prev 指针指向待交换的前一个节点（初始为 dummy）
        ListNode prev = dummy;

        // 只要还存在两个节点，就能交换
        while (prev.next != null && prev.next.next != null) {
            // 取出要交换的两个节点
            ListNode first = prev.next;
            ListNode second = first.next;

            // 开始交换
            first.next = second.next;  // 把 first 指向 second 后面的节点
            second.next = first;       // 把 second 指向 first，实现反转
            prev.next = second;        // 把前驱 prev 指向 second，连接上交换后的部分

            // 移动 prev 到下一对前面
            prev = first;
        }

        // 返回新的头节点
        return dummy.next;
    }
}
```

## 🧩执行过程图示（可脑补或画图）

假设输入：

```
1 -> 2 -> 3 -> 4
```

操作过程：

| 步骤      | 操作                  | 链表状态  |
| --------- | --------------------- | --------- |
| 初始      | dummy → 1 → 2 → 3 → 4 |           |
| 交换 1、2 | 2 → 1 → 3 → 4         |           |
| 交换 3、4 | 2 → 1 → 4 → 3         | ✅最终结果 |

## 💬面试标准回答模板

> 我使用虚拟头节点 + 三指针法。
>  每次取出相邻两个节点 `first` 和 `second`，调整三条 `next` 引用，实现局部反转。
>  dummy 节点用来统一处理头节点被交换的情况。
>  整体时间复杂度 O(n)，空间复杂度 O(1)。
