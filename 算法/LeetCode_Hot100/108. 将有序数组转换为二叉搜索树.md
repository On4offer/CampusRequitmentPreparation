------

## 🧩 一、题意理解

> 给定一个 **升序排序的整型数组 `nums`**，将其转换成一棵 **高度平衡的二叉搜索树（BST）**。
>
> “高度平衡”指的是：
>  任意节点的左右子树高度差不超过 1。

------

### 示例

输入：

```
nums = [-10, -3, 0, 5, 9]
```

输出（可能的结果之一）：

```
       0
     /   \
   -3     9
   /     /
 -10    5
```

------

## 💡 二、思路分析

**二叉搜索树 (BST) 特性：**

> 对于任意节点，左子树所有节点 < 根节点 < 右子树所有节点。

**有序数组的中间元素**可以作为根节点：

- 左半部分 → 左子树
- 右半部分 → 右子树
- 递归构建即可。

这样自然保证树**平衡**，因为每次都取中点作为根。

------

## 🔁 三、递归思路

递归函数定义：

```
build(nums, left, right):
    若 left > right → 返回 null
    mid = (left + right) / 2
    node = new TreeNode(nums[mid])
    node.left = build(nums, left, mid - 1)
    node.right = build(nums, mid + 1, right)
    返回 node
```

------

## ✅ 四、Java 实现（递归法）

```java
class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        return build(nums, 0, nums.length - 1);
    }

    private TreeNode build(int[] nums, int left, int right) {
        if (left > right) return null; // 递归终止条件

        int mid = left + (right - left) / 2; // 防止溢出
        TreeNode root = new TreeNode(nums[mid]); // 取中点为根

        root.left = build(nums, left, mid - 1);  // 构建左子树
        root.right = build(nums, mid + 1, right); // 构建右子树

        return root;
    }
}
```

------

## 🔄 五、执行流程示例

以 `nums = [-10, -3, 0, 5, 9]` 为例：

| 调用层级 | 区间 [left,right] | mid  | 创建节点 | 递归     |
| -------- | ----------------- | ---- | -------- | -------- |
| 1        | [0,4]             | 2    | 0        | 构建左右 |
| 2        | [0,1]             | 0    | -10      | 构建右   |
| 3        | [1,1]             | 1    | -3       | 返回     |
| 2        | [3,4]             | 3    | 5        | 构建右   |
| 3        | [4,4]             | 4    | 9        | 返回     |

最终生成的树：

```
        0
      /   \
   -10     5
     \       \
     -3       9
```

（不同中点选择方式可能略有不同，但都满足 BST + 平衡）

## 💬 七、标准面试回答模板

> 这题考察的是二叉搜索树和递归分治的理解。
>  我每次取数组中点作为根节点，左右两半分别构建左右子树。
>  这样可以保证平衡性。
>  时间复杂度 O(n)，空间复杂度 O(log n)。

------

## 🔍 八、常见追问

| 追问                     | 答法                                                     |
| ------------------------ | -------------------------------------------------------- |
| 为什么取中点能保证平衡？ | 因为左右子数组长度相差不超过 1，对应子树高度差不超过 1。 |
| 如果数组是降序的怎么办？ | 先反转或直接取反序索引。                                 |
| 可以不用递归实现吗？     | 可以用栈或队列手动模拟构建过程（较复杂）。               |
