# 🧩 LeetCode 114. Flatten Binary Tree to Linked List

------

## 📘题意说明

给定一个二叉树，将其“压平”成一个单链表。

要求：

- 链表的顺序应当是**先序遍历**的顺序（即根 -> 左子树 -> 右子树）。
- 不要创建新的节点，**只能修改树中的指针**。

**示例：**

```
输入：
     1
    / \
   2   5
  / \   \
 3   4   6

输出：
1 -> 2 -> 3 -> 4 -> 5 -> 6
```

------

## 🧠思路与指针设计

这题是二叉树的经典题目，核心思想是**递归和原地修改指针**。

### ✳️ 核心思想：

1. 将二叉树展开成链表，关键在于调整树节点的指针。
2. 先递归遍历左右子树，将左子树连接到当前节点的右边，右子树则接到其后的节点。
3. **遍历顺序：我们需要按先序遍历**进行调整：先访问根节点，再处理左子树，最后是右子树。

### ✳️ 关键指针设计：

- `root` 当前节点，指向正在展开的节点。
- `left` 左子树的根节点。
- `right` 右子树的根节点。

------

## 🧾 步骤伪代码

```
1. 如果当前节点为空，则返回。
2. 如果当前节点有左子树：
   a. 先递归展开左子树
   b. 将左子树接到当前节点的右边
   c. 将右子树接到左子树的末端
3. 递归展开右子树
4. 返回根节点
```

## 💻 Java 标准实现

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public void flatten(TreeNode root) {
        if (root == null) return;
        
        // 递归展平左子树和右子树
        flatten(root.left);
        flatten(root.right);

        // 如果左子树不为空，将左子树接到右边
        if (root.left != null) {
            TreeNode tempRight = root.right;  // 暂存右子树
            root.right = root.left;           // 左子树接到右边
            root.left = null;                 // 左子树置空

            // 找到左子树的最右节点，接上右子树
            TreeNode temp = root.right;
            while (temp.right != null) {
                temp = temp.right;
            }
            temp.right = tempRight;
        }
    }
}
```

## 💬 面试回答模板

> **思路：**
>  使用递归方法遍历树节点，按照先序遍历的顺序修改节点指针。递归展开左右子树，左子树连接到右子树，右子树连接到左子树末尾。
>
> **关键点：**
>
> - 使用递归展平左、右子树；
> - 将左子树接到右边，并将右子树接到左子树的末尾。
> - 树的指针需要原地修改，不能新建节点。
>
> **复杂度：**
>  时间复杂度 O(n)，空间复杂度 O(1)。

## ⚠️ 常见坑点

1. `root.left = null;` 不要忘记将左子树指针置空，否则会导致指针错乱。
2. 在连接左子树和右子树时，必须**先找到左子树的最右节点**，否则无法正确连接右子树。
3. 注意递归结束条件，要确保递归到叶子节点时返回。
