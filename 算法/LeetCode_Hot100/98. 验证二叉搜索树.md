这是一个经典面试题：**验证二叉搜索树（Validate Binary Search Tree）**。
 我们来按照你指定的面试题讲解模板逐步拆解👇

------

## 🧩 一、题意理解

> **题目：** 给你一个二叉树的根节点 `root`，判断它是否是一个 **二叉搜索树（BST）**。

**二叉搜索树（BST）定义：**

- 左子树所有节点的值 **小于** 根节点的值；
- 右子树所有节点的值 **大于** 根节点的值；
- 左右子树也必须分别是二叉搜索树。

------

## 💡 二、思路分析

验证BST有两种主流思路：

### ✅ 方法1：中序遍历（最常见）

- **性质：** 二叉搜索树的中序遍历结果是**严格递增**的。
- **思路：**
  1. 对整棵树进行中序遍历；
  2. 记录上一个访问的节点值；
  3. 如果当前节点 ≤ 上一个节点，说明不是BST。

👉 时间复杂度 O(n)，空间复杂度 O(n)（递归栈）。

------

### ✅ 方法2：递归上下界判断（更直观）

- **思路：**
  - 递归每个节点时，维护一个**数值范围 (min, max)**；
  - 当前节点必须满足 `min < node.val < max`；
  - 左子树的范围是 `(min, node.val)`；
  - 右子树的范围是 `(node.val, max)`；
  - 如果任何节点不满足条件，则返回 false。

👉 时间复杂度 O(n)，空间复杂度 O(h)。

------

## 🧠 三、关键伪代码

```java
boolean isValidBST(TreeNode node, long min, long max) {
    if (node == null) return true;
    if (node.val <= min || node.val >= max) return false;
    return isValidBST(node.left, min, node.val)
        && isValidBST(node.right, node.val, max);
}
```

------

## 🧩 四、Java 实现代码（推荐写法）

```java
class Solution {
    public boolean isValidBST(TreeNode root) {
        return helper(root, Long.MIN_VALUE, Long.MAX_VALUE);
    }

    private boolean helper(TreeNode node, long min, long max) {
        if (node == null) return true;
        if (node.val <= min || node.val >= max) return false;
        return helper(node.left, min, node.val) &&
               helper(node.right, node.val, max);
    }
}
```

> 注意这里使用 `long`，避免节点值为 `Integer.MIN_VALUE` 或 `Integer.MAX_VALUE` 时出现溢出错误。

## 🌳 五、图解示例

以如下树为例：

```
      5
     / \
    3   7
   / \   \
  2  4    8
```

遍历过程：

- `5` 在区间 `(-∞, +∞)` ✅
- 左子树 `3` 在 `(-∞, 5)` ✅
- 右子树 `7` 在 `(5, +∞)` ✅
- 所有节点满足条件 → 返回 `true`

## 🎯 七、标准面试回答模板

> **面试官问：如何判断一棵二叉树是否是BST？**

我会使用递归的上下界判断法。
 对于每个节点，维护其允许的取值范围 `(min, max)`，
 当节点值不在这个范围时返回 false。
 左右子树递归时更新范围：左子树 `(min, node.val)`，右子树 `(node.val, max)`。
 最终时间复杂度为 O(n)，空间复杂度为 O(h)。

## 🔍 八、可扩展追问（面试官可能问）

1. **为什么不能只判断左右节点大小？**
    因为左子树的所有节点都要小于根节点，而不仅是左孩子。
2. **为什么用 long 而不是 int？**
    防止极端值溢出。
3. **能用中序遍历实现吗？**
    可以，用一个变量记录前一个节点的值，保证中序结果严格递增。
4. **BST 和完全二叉树、平衡二叉树的区别？**
   - BST：左小右大。
   - 完全二叉树：节点尽量靠左排列。
   - 平衡二叉树（AVL/红黑树）：左右子树高度差不超过1。
