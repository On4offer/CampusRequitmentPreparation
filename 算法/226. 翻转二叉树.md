这是 LeetCode 热题「226. 翻转二叉树 (Invert Binary Tree)」——面试中很常见的一道基础题。下面我给你一个完整的**面试题模板式讲解** 👇

------

## 🧩 一、题意理解

> **题目**：将一棵二叉树左右子树全部翻转。
>  **输入**：二叉树根节点 `root`
>  **输出**：翻转后的根节点

**示例：**

```
输入：
     4
   /   \
  2     7
 / \   / \
1  3  6  9

输出：
     4
   /   \
  7     2
 / \   / \
9  6  3  1
```

------

## 💡 二、思路分析

翻转二叉树的核心是：

> 对每个节点，将它的左右子节点交换。

可以通过三种方式实现：

1. **递归（最常见）**
2. **层序遍历（BFS）**
3. **栈模拟递归（DFS）**

------

## 🧠 三、递归思路（面试首选）

递归公式：

```
invertTree(root):
    1. 交换左右子树
    2. 递归翻转左子树
    3. 递归翻转右子树
```

------

## 🧩 四、伪代码

```
if root == null:
    return null

swap(root.left, root.right)
invertTree(root.left)
invertTree(root.right)

return root
```

------

## ✅ 五、Java实现（递归）

```java
class Solution {
    public TreeNode invertTree(TreeNode root) {
        // 1. 终止条件
        if (root == null) return null;

        // 2. 交换左右子节点
        TreeNode temp = root.left;
        root.left = root.right;
        root.right = temp;

        // 3. 递归处理左右子树
        invertTree(root.left);
        invertTree(root.right);

        // 4. 返回根节点
        return root;
    }
}
```

## ⚙️ 七、复杂度分析

- **时间复杂度**：`O(n)` —— 每个节点访问一次
- **空间复杂度**：`O(h)` —— 递归栈深度，最坏为 `O(n)`（链式树），平均 `O(log n)`

## 🧱 八、迭代实现（可作为追问回答）

```java
class Solution {
    public TreeNode invertTree(TreeNode root) {
        if (root == null) return null;
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);

        while (!queue.isEmpty()) {
            TreeNode node = queue.poll();
            // 交换左右
            TreeNode temp = node.left;
            node.left = node.right;
            node.right = temp;

            // 入队左右子节点
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        return root;
    }
}
```

## 💬 九、面试回答模板

> 这题考察的是二叉树的遍历与递归思维。
>
> 我采用递归法实现。对于每个节点，先交换它的左右子树，然后递归翻转左右子树即可。时间复杂度是 O(n)，空间复杂度取决于递归深度 O(h)。
>  另外也可以用队列进行层序遍历来实现迭代版本。
