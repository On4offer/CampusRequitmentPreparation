很好👏！这道题是**二叉树的最大深度**（LeetCode 104），非常经典。
 我们来按照面试模板一步步讲解。

------

## 🧩 一、题意理解

给定一棵二叉树，返回它的**最大深度**。

> 深度：从根节点到最远叶子节点的最长路径上**节点的数量**。

------

### 🌳 举个例子：

```
      3
     / \
    9  20
       / \
      15  7
```

👉 这棵树的最大深度是 `3`
 （路径为 3 → 20 → 15 或 3 → 20 → 7）。

------

## 🧠 二、思路分析

### ✅ 递归思路（最常见）

“最大深度”其实就是：

> 1 + 左子树的最大深度 和 右子树的最大深度 的较大值。

也就是说：

- 如果根节点为空，深度是 `0`；
- 否则，递归求左右子树深度，取较大值，再 +1。

------

## 💻 三、递归写法（模板）

```java
class Solution {
    public int maxDepth(TreeNode root) {
        if (root == null) return 0;  // 递归出口：空节点深度为0

        int leftDepth = maxDepth(root.left);   // 左子树的最大深度
        int rightDepth = maxDepth(root.right); // 右子树的最大深度

        return Math.max(leftDepth, rightDepth) + 1; // 当前节点的深度
    }
}
```

------

## 🔍 四、执行逻辑（逐步说明）

假设输入树：

```
      1
     / \
    2   3
   /
  4
```

执行过程👇

| 调用          | 左子树深度 | 右子树深度 | 返回值 |
| ------------- | ---------- | ---------- | ------ |
| `maxDepth(4)` | 0          | 0          | 1      |
| `maxDepth(2)` | 1          | 0          | 2      |
| `maxDepth(3)` | 0          | 0          | 1      |
| `maxDepth(1)` | 2          | 1          | 3      |

最终返回 **3**。

## 💻 一、完整代码（迭代版 BFS）

```java
import java.util.*;

class Solution {
    public int maxDepth(TreeNode root) {
        if (root == null) return 0;  // 空树深度为 0

        Queue<TreeNode> queue = new LinkedList<>();  // 队列用于层序遍历
        queue.offer(root);   // 根节点入队
        int depth = 0;       // 层数计数器

        // 每一轮 while 表示“遍历一层”
        while (!queue.isEmpty()) {
            int size = queue.size();  // 当前层节点个数
            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll(); // 出队当前节点
                if (node.left != null) queue.offer(node.left);   // 左子树入队
                if (node.right != null) queue.offer(node.right); // 右子树入队
            }
            depth++;  // 当前层遍历完，层数+1
        }

        return depth;
    }
}
```

------

## 🧠 二、思路讲解（BFS 层序遍历）

我们一层一层遍历整棵树，
 每遍历一层，说明树的深度又加一层。

> BFS = Breadth First Search
>  → 宽度优先遍历，也叫层序遍历。

------

## 🌳 三、例子图解

假设这棵树：

```
      3
     / \
    9  20
       / \
      15  7
```

执行过程👇

| 阶段   | 队列内容 | 操作                  | depth |
| ------ | -------- | --------------------- | ----- |
| 初始化 | [3]      | 根节点入队            | 0     |
| 第一层 | [9, 20]  | 处理 3 → 加入 9,20    | 1     |
| 第二层 | [15, 7]  | 处理 9,20 → 加入 15,7 | 2     |
| 第三层 | []       | 处理 15,7 → 无子节点  | 3     |

队列为空 → 遍历结束
 返回 **depth = 3**

## ⚙️ 四、关键逻辑逐行解释

| 行号                                          | 代码                    | 说明 |
| --------------------------------------------- | ----------------------- | ---- |
| `if (root == null) return 0;`                 | 空树直接返回 0          |      |
| `Queue<TreeNode> queue = new LinkedList<>();` | 用队列存放每层节点      |      |
| `queue.offer(root);`                          | 根节点入队              |      |
| `int depth = 0;`                              | 初始化层数计数器        |      |
| `while (!queue.isEmpty()) { ... }`            | 一层层遍历              |      |
| `int size = queue.size();`                    | 当前层节点数量          |      |
| `for (int i = 0; i < size; i++)`              | 遍历当前层的所有节点    |      |
| `queue.poll()`                                | 从队列取出节点          |      |
| `queue.offer(node.left/right)`                | 把下一层节点入队        |      |
| `depth++`                                     | 当前层遍历完 → 深度加一 |      |
