## 🧩 一、题意说明

> 给你一个链表，要求在 **O(n log n)** 时间内排序，并且只使用 **常数级额外空间**。

------

## 🧠 二、思路分析（归并排序）

链表的随机访问效率很低（无法用快排的“随机下标”思想），
 最适合的排序算法是 **归并排序（Merge Sort）**。

归并排序核心思想：

1. **分割（Divide）**：用快慢指针找到中点，把链表一分为二；
2. **递归排序（Conquer）**：分别对左右两半递归排序；
3. **合并（Merge）**：将两个有序链表合并为一个有序链表。

------

## 🎯 三、指针设计

| 指针名          | 含义                      |
| --------------- | ------------------------- |
| `slow`          | 慢指针，用于找链表中点    |
| `fast`          | 快指针，移动两倍速度      |
| `mid`           | 链表分割点（`slow.next`） |
| `l1`、`l2`      | 左右两部分链表            |
| `dummy`、`tail` | 用于合并阶段              |

------

## 🧾 四、步骤伪代码

```
sortList(head):
    if head == null or head.next == null:
        return head

    // 1. 找中点
    slow, fast = head, head.next
    while fast != null and fast.next != null:
        slow = slow.next
        fast = fast.next.next
    mid = slow.next
    slow.next = null   // 断开两部分

    // 2. 递归排序左右两半
    left = sortList(head)
    right = sortList(mid)

    // 3. 合并两个有序链表
    return merge(left, right)
```

## 💻 五、Java标准实现

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode sortList(ListNode head) {
        // 递归出口
        if (head == null || head.next == null) return head;

        // 1️⃣ 快慢指针找中点
        ListNode slow = head, fast = head.next;
        while (fast != null && fast.next != null) {
            slow = slow.next;
            fast = fast.next.next;
        }
        ListNode mid = slow.next;
        slow.next = null; // 断开链表

        // 2️⃣ 递归排序两部分
        ListNode left = sortList(head);
        ListNode right = sortList(mid);

        // 3️⃣ 合并两个有序链表
        return merge(left, right);
    }

    // 合并两个有序链表
    private ListNode merge(ListNode l1, ListNode l2) {
        ListNode dummy = new ListNode(0);
        ListNode tail = dummy;

        while (l1 != null && l2 != null) {
            if (l1.val <= l2.val) {
                tail.next = l1;
                l1 = l1.next;
            } else {
                tail.next = l2;
                l2 = l2.next;
            }
            tail = tail.next;
        }

        // 拼接剩余节点
        tail.next = (l1 != null) ? l1 : l2;
        return dummy.next;
    }
}
```

## 🧩 六、图解说明（链表分割 + 合并）

**示例：**

```
4 → 2 → 1 → 3
```

1️⃣ 找中点 → 分割成：

```
[4 → 2]    [1 → 3]
```

2️⃣ 递归排序各自：

```
[2 → 4]    [1 → 3]
```

3️⃣ 合并：

```
1 → 2 → 3 → 4
```

## 💬 八、面试回答模板

> **思路**：
>  使用归并排序。链表无法随机访问，因此不适合快排。
>  用快慢指针找到中点将链表一分为二，递归地分别排序，再通过有序链表合并函数合并结果。
>
> **关键点：**
>
> - 用快慢指针找中点；
> - 注意断开链表；
> - 合并时用虚拟头节点 dummy 简化操作。
>
> **复杂度：**
>  时间 O(n log n)，空间 O(1)。

------

## ⚠️ 九、常见坑

1. **忘记断开链表**（`slow.next = null`），导致死循环；
2. 合并时漏拼剩余部分；
3. 快慢指针初始化错误（`fast = head.next` 避免死循环）；
4. 若要写非递归版，可使用“自底向上归并”，面试中说思路即可。
