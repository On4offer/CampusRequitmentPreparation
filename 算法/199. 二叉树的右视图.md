### **LeetCode 第199题：二叉树的右视图**

------

## 💡题目要求

> 给定一个二叉树，想象自己站在树的右侧，返回树的右视图。

**示例：**

```
输入：[1,2,3,null,5,null,4]
输出：[1, 3, 4]
解释:
   1            <--- 视图是从右侧看到的
 /   \
2     3         <--- 3 是右侧的可见节点
 \     \
  5     4       <--- 4 是右侧的可见节点
```

------

## 🧠 思路分析

### 1️⃣ **核心思路**：

- 对于每一层，我们从右到左遍历，找到该层的第一个节点（就是我们从右视图看到的节点）。
- 我们可以使用 **层序遍历**（广度优先遍历，BFS）来遍历二叉树。
- 在每一层的遍历中，**首先访问右子树**，然后访问左子树。这样，我们在层序遍历时第一个访问到的节点就是该层的最右侧节点。
- 需要一个队列来存储节点，队列实现层序遍历。

### 2️⃣ **具体步骤**：

- 使用 **队列** 来进行广度优先遍历（BFS）；
- 每次遍历一层时，**从右到左遍历**，保证每层的第一个访问节点就是该层的最右侧节点；
- 对于每一层的节点，只保存 **该层的第一个节点**，这就是右视图的节点。

------

## ✅ 代码实现

```java
import java.util.*;

class Solution {
    public List<Integer> rightSideView(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        if (root == null) return result;  // 如果根节点为空，直接返回空列表
        
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);  // 把根节点加入队列
        
        while (!queue.isEmpty()) {
            int levelSize = queue.size();  // 当前层的节点数
            
            // 遍历当前层的所有节点
            for (int i = 0; i < levelSize; i++) {
                TreeNode currentNode = queue.poll();
                
                // 如果是该层的第一个节点，添加到结果中
                if (i == 0) {
                    result.add(currentNode.val);
                }
                
                // 把当前节点的右子树和左子树加入队列（右子树先加入）
                if (currentNode.right != null) {
                    queue.offer(currentNode.right);
                }
                if (currentNode.left != null) {
                    queue.offer(currentNode.left);
                }
            }
        }
        
        return result;
    }
}
```

## 🧩 复杂度分析

| 项目       | 复杂度                                     |
| ---------- | ------------------------------------------ |
| 时间复杂度 | O(N)（N 为节点数）                         |
| 空间复杂度 | O(N)（最坏情况下，队列中可能存储所有节点） |

- **时间复杂度**：`O(N)`，我们需要遍历树中的每个节点一次，每个节点入队一次并出队一次。
- **空间复杂度**：`O(N)`，队列在最坏情况下可能存储树中所有的节点（例如满二叉树的最后一层）。

## 💬 面试官可能追问

1. **为什么从右到左遍历每一层？**
   - 因为我们要得到每一层的最右侧节点，所以在层序遍历时，应该优先访问右子树。
2. **能否不使用队列进行层序遍历？**
   - 队列是 BFS 中常用的数据结构，它可以有效地实现层序遍历。尽管可以使用递归来模拟层序遍历，但队列是更直观且高效的做法。
3. **如果树只有一条直线（例如只有右子树或左子树），结果会是什么样子？**
   - 如果树只有一条直线，那么每一层的右视图就是当前节点本身。例如，如果只有右子树，右视图会返回每一层的节点值。

## 🧱 总结模板（可背诵）

> **面试题：二叉树的右视图**
>  **思路：** 使用广度优先遍历（BFS），每一层从右到左遍历，记录每一层的第一个节点。
>  **关键技巧：** 用队列进行层序遍历，确保每层的第一个节点是右侧可见节点。
>  **时间复杂度 O(N)，空间复杂度 O(N)。**
