# 🧩 LeetCode 25. K 个一组反转链表（Reverse Nodes in k-Group）

------

## 📘题意说明

给定一个链表 `head`，将链表每 `k` 个节点一组进行反转。
 若最后一组节点数不足 `k`，则保持原顺序不变。

**示例：**

```
输入：1 -> 2 -> 3 -> 4 -> 5, k = 2
输出：2 -> 1 -> 4 -> 3 -> 5
输入：1 -> 2 -> 3 -> 4 -> 5, k = 3
输出：3 -> 2 -> 1 -> 4 -> 5
```

------

## 🧠思路与指针设计

这题是“**两两交换节点（swapPairs）**”的升级版。
 关键在于：**分组反转 + 前后连接**。

### ✳️ 核心思想：

- 使用一个 **虚拟头节点 dummy**，方便处理头部。
- 每次从 `prev` 开始，向后找 `k` 个节点作为一组；
- 如果数量不足 `k`，直接返回；
- 否则，反转当前组并连接回主链表。

### ✳️ 三个关键指针：

| 指针    | 含义                                        |
| ------- | ------------------------------------------- |
| `prev`  | 当前组反转前的前驱节点                      |
| `start` | 当前组的第一个节点                          |
| `end`   | 当前组的最后一个节点（用来判断够不够 k 个） |

------

## 🧾 步骤伪代码

```
1. 创建 dummy 节点指向 head
2. prev = dummy
3. while True:
       end = prev
       往后移动 k 次找到组尾
       若 end == null → break
       nextGroup = end.next
       start = prev.next
       end.next = null
       反转 start~end 区间
       prev.next = 反转后新头
       start.next = nextGroup
       prev = start
4. 返回 dummy.next
```

------

## 💻 Java 标准实现

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode reverseKGroup(ListNode head, int k) {
        // 虚拟头节点，简化操作
        ListNode dummy = new ListNode(0, head);
        ListNode prev = dummy;

        while (true) {
            // 找到当前组的尾节点
            ListNode end = prev;
            for (int i = 0; i < k && end != null; i++) {
                end = end.next;
            }
            if (end == null) break; // 不足 k 个，退出

            ListNode nextGroup = end.next; // 下一组起点
            ListNode start = prev.next;    // 当前组起点
            end.next = null;               // 断开当前组

            // 反转 [start, end] 区间
            ListNode newHead = reverse(start);

            // 重新连接
            prev.next = newHead;
            start.next = nextGroup;

            // prev 移动到下一组前
            prev = start;
        }

        return dummy.next;
    }

    // 标准链表反转函数
    private ListNode reverse(ListNode head) {
        ListNode prev = null;   // 前驱指针
        ListNode cur = head;    // 当前节点指针
        while (cur != null) {
            ListNode next = cur.next;  // 暂存后继
            cur.next = prev;           // 改方向
            prev = cur;                // 前移 prev
            cur = next;                // 前移 cur
        }
        return prev;                   // 返回新头
    }
}
```

## 🧩 图解说明（以 k=3 举例）

初始链表：

```
dummy → 1 → 2 → 3 → 4 → 5
```

操作步骤：

1. 找到 `[1,2,3]` → 反转为 `3 → 2 → 1`
2. 重新连接：

```
dummy → 3 → 2 → 1 → 4 → 5
```

1. 下一组 `[4,5]` 不够 k=3 → 停止。

最终结果：

```
3 → 2 → 1 → 4 → 5
```

## 💬 面试回答模板

> **思路：**
>  使用虚拟头节点 dummy，每次定位 k 个节点一组。
>  若不足 k 个则保持不动，否则反转该段并重新连接。
>
> **关键点：**
>
> - 用 end 指针判断是否够 k 个；
> - 翻转时先断开，结束后再重新连接；
> - 注意更新 prev 到新尾节点。
>
> **复杂度：**
>  O(n) 时间，O(1) 空间。

------

## ⚠️ 常见坑点

1. `end` 指针循环 `k` 次要判断 `end != null`，否则 NPE。
2. `end.next = null` 必须断开，否则反转会串链。
3. 翻转函数要返回新头；反转后原 `start` 成为新尾。
4. 最后一组不满 `k` 时直接跳出循环，保持原样。
