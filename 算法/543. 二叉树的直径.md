这是 LeetCode 热题 **543. 二叉树的直径（Diameter of Binary Tree）** —— 高频经典面试题。
 下面给出完整的 **面试题结构化讲解模板** 👇

------

## 🧩 一、题意理解

> 给你一棵二叉树，**返回它的直径长度**。
>
> 直径：任意两个节点之间的最长路径长度（以边数表示）。

------

### 示例：

```
输入：
     1
   /   \
  2     3
 / \
4   5

输出：3
解释：最长路径为 4 → 2 → 1 → 3 或 5 → 2 → 1 → 3，长度为 3。
```

------

## 💡 二、思路分析

直径路径可以经过根节点，也可以不经过根节点。

👉 **核心思想**：

- 对每个节点，计算：
  - 左子树深度 `L`
  - 右子树深度 `R`
  - 经过该节点的最长路径 = `L + R`
- 所有节点中最大的 `L + R` 即为整棵树的直径。

------

### 🔁 递归思路

1. 定义函数 `depth(node)`：返回从该节点出发的最大深度；
2. 在递归过程中同时维护一个全局变量 `maxDiameter`；
3. 对每个节点更新：
    `maxDiameter = max(maxDiameter, leftDepth + rightDepth)`。

------

## 🧩 三、伪代码

```
maxDiameter = 0

function depth(node):
    if node == null: return 0
    left = depth(node.left)
    right = depth(node.right)
    maxDiameter = max(maxDiameter, left + right)
    return 1 + max(left, right)

diameterOfBinaryTree(root):
    depth(root)
    return maxDiameter
```

------

## ✅ 四、Java 实现（递归版）

```java
class Solution {
    private int maxDiameter = 0; // 记录最大直径

    public int diameterOfBinaryTree(TreeNode root) {
        dfs(root);
        return maxDiameter;
    }

    private int dfs(TreeNode node) {
        if (node == null) return 0;

        // 递归计算左右子树深度
        int leftDepth = dfs(node.left);
        int rightDepth = dfs(node.right);

        // 更新最大直径
        maxDiameter = Math.max(maxDiameter, leftDepth + rightDepth);

        // 返回当前节点深度
        return 1 + Math.max(leftDepth, rightDepth);
    }
}
```

## 🔄 五、执行流程演示

以如下树为例：

```
     1
   /   \
  2     3
 / \
4   5
```

| 节点 | 左深度 | 右深度 | left+right | 当前maxDiameter | 返回深度 |
| ---- | ------ | ------ | ---------- | --------------- | -------- |
| 4    | 0      | 0      | 0          | 0               | 1        |
| 5    | 0      | 0      | 0          | 0               | 1        |
| 2    | 1      | 1      | 2          | 2               | 2        |
| 3    | 0      | 0      | 0          | 2               | 1        |
| 1    | 2      | 1      | 3          | **3**           | 3        |

最终输出：`3`

## ⚙️ 六、复杂度分析

| 项目       | 分析                        |
| ---------- | --------------------------- |
| 时间复杂度 | O(n)，每个节点遍历一次      |
| 空间复杂度 | O(h)，递归栈深度（h为树高） |

------

## 💬 七、标准面试回答模板

> 这题考察的是二叉树递归的综合应用。
>
> 我使用深度优先搜索（DFS），在求每个节点深度的同时，记录经过该节点的最大路径长度。
>  通过维护一个全局变量不断更新最大值即可。
>  时间复杂度为 O(n)，空间复杂度为 O(h)。

## 🔍 八、面试追问方向

| 追问方向 | 面试官可能问什么                         |
| -------- | ---------------------------------------- |
| 理解     | 为什么不是返回 max(left, right)+1？      |
| 延伸     | 如何返回**直径对应的两个节点**？         |
| 实战     | 如果树节点是N叉树该怎么改？              |
| 调试     | 空树、单节点树的返回值是多少？（应为 0） |

------

## 🧠 九、易错点总结

| 错误                             | 原因                               |
| -------------------------------- | ---------------------------------- |
| ❌ 把直径当作节点数               | 实际上是**边数**，所以不加1        |
| ❌ 忘记更新全局变量               | `maxDiameter` 必须定义在函数外部   |
| ❌ 在递归返回时更新而不是中途更新 | 必须在回溯时计算左右子树后更新直径 |
