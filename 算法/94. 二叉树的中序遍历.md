我们来系统讲解这道题（中序遍历二叉树）👇

------

## 🧩 一、题意

给定一棵二叉树，返回它的**中序遍历**结果（左 → 根 → 右）。

------

## 🧠 二、思路分析

中序遍历有三种常见写法：

### ✅ 1️⃣ 递归写法（最简单）

利用函数调用栈实现“左 → 根 → 右”访问顺序。

### ✅ 2️⃣ 迭代写法（面试常问）

显式用 **栈** 模拟递归过程。

------

## 💻 三、递归写法（模板解法）

```java
class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        dfs(root, res);
        return res;
    }

    private void dfs(TreeNode node, List<Integer> res) {
        if (node == null) return;
        dfs(node.left, res);      // 1. 遍历左子树
        res.add(node.val);        // 2. 访问当前节点
        dfs(node.right, res);     // 3. 遍历右子树
    }
}
```

------

## ⚙️ 四、迭代写法（手动模拟递归栈）

```java
class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        Stack<TreeNode> stack = new Stack<>();
        TreeNode cur = root;

        while (cur != null || !stack.isEmpty()) {
            // 一直向左走，直到最左节点
            while (cur != null) {
                stack.push(cur);
                cur = cur.left;
            }

            // 弹出栈顶节点，访问它
            cur = stack.pop();
            res.add(cur.val);

            // 转向右子树
            cur = cur.right;
        }

        return res;
    }
}
```

------

## 📊 五、复杂度分析

- **时间复杂度**：O(n)
   每个节点访问一次。
- **空间复杂度**：O(n)
   最坏情况下栈深度为 n（单链树）。

## 🧭 六、图解说明

例如树结构：

```
    1
     \
      2
     /
    3
```

中序遍历过程：

```
访问顺序： [1, 3, 2]
```

执行轨迹：
 1️⃣ 走到最左 → 节点 1
 2️⃣ 弹出访问 1 → 右走到 2 的左
 3️⃣ 访问 3 → 回到 2
 4️⃣ 访问 2 → 结束

> 面试官：说说中序遍历怎么实现？

> ✅ 我：

>
> - 递归写法简单，但栈深度受限。
> - 迭代写法显式用栈保存节点，更直观。
>    时间复杂度是 O(n)，空间复杂度 O(n)。

## 🧩 一、代码结构解释

这部分是题目给定的二叉树节点定义（LeetCode 通用格式）：

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
```

------

## 🧠 二、逐行解释

| 行号                                               | 代码                                                 | 含义 |
| -------------------------------------------------- | ---------------------------------------------------- | ---- |
| `public class TreeNode {`                          | 定义一个**二叉树节点类**，类名为 `TreeNode`          |      |
| `int val;`                                         | 节点存储的**数值**（比如 1、2、3）                   |      |
| `TreeNode left;`                                   | 指向**左子节点**的引用（没有则为 null）              |      |
| `TreeNode right;`                                  | 指向**右子节点**的引用（没有则为 null）              |      |
| `TreeNode() {}`                                    | **无参构造函数**，默认创建空节点                     |      |
| `TreeNode(int val) { this.val = val; }`            | **单参数构造函数**，直接给节点赋值                   |      |
| `TreeNode(int val, TreeNode left, TreeNode right)` | **三参数构造函数**，可一次性创建节点并指定左右子节点 |      |
| `}`                                                | 结束类定义                                           |      |

## 🌳 三、举例说明

我们用代码手动创建一棵简单二叉树：

```
      1
     / \
    2   3
```

对应的 Java 代码是：

```java
TreeNode left = new TreeNode(2);
TreeNode right = new TreeNode(3);
TreeNode root = new TreeNode(1, left, right);
```

- `new TreeNode(2)` 创建值为 2 的左节点；
- `new TreeNode(3)` 创建值为 3 的右节点；
- `new TreeNode(1, left, right)` 创建根节点 1，并连接左右子树。

------

## 💡 四、整体理解

这段定义其实就是：

> 用一个类 `TreeNode` 来描述“树上的一个节点”，
>  每个节点包含：
>
> - **一个值**（`val`）
> - **左子节点引用**（`left`）
> - **右子节点引用**（`right`）

多个节点互相引用（left/right）后，就构成了一棵完整的二叉树结构。
