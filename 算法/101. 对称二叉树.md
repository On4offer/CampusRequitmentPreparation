这是 LeetCode 热题 **101. 对称二叉树 (Symmetric Tree)**，属于二叉树遍历与递归基础题，非常常见于校招 Java 后端笔试/面试。
 我们用**面试标准讲解模板**为你拆解 👇

------

## 🧩 一、题意理解

> 判断一棵二叉树是否关于中心轴对称。

### 示例：

```
输入：
     1
   /   \
  2     2
 / \   / \
3  4  4  3

输出：true
输入：
     1
   /   \
  2     2
   \     \
    3     3

输出：false
```

------

## 💡 二、思路分析

**对称二叉树**的定义是：

> 对于任意节点 `root`，它的左子树 `left` 与右子树 `right` 互为镜像。

也就是说：

- 左子树的左节点 ↔ 右子树的右节点；
- 左子树的右节点 ↔ 右子树的左节点。

------

### 🔁 递归思路（面试首选）

判断两个子树是否镜像：

```
isMirror(p, q):
    若 p、q 均为空 → true
    若 p 或 q 为空 → false
    若 p.val != q.val → false
    否则：
        比较 p.left vs q.right && p.right vs q.left
```

整体思路：

```
isSymmetric(root):
    若 root 为空 → true
    否则调用 isMirror(root.left, root.right)
```

------

## 🧩 三、伪代码

```
if root == null:
    return true
return isMirror(root.left, root.right)

function isMirror(t1, t2):
    if t1 == null && t2 == null: return true
    if t1 == null || t2 == null: return false
    return (t1.val == t2.val)
           && isMirror(t1.left, t2.right)
           && isMirror(t1.right, t2.left)
```

------

## ✅ 四、Java 实现（递归版）

```java
class Solution {
    public boolean isSymmetric(TreeNode root) {
        if (root == null) return true;
        return isMirror(root.left, root.right);
    }

    private boolean isMirror(TreeNode t1, TreeNode t2) {
        if (t1 == null && t2 == null) return true;  // 都为空，对称
        if (t1 == null || t2 == null) return false; // 一空一不空，不对称
        if (t1.val != t2.val) return false;         // 值不同，不对称

        // 递归比较：外侧 vs 内侧
        return isMirror(t1.left, t2.right) && isMirror(t1.right, t2.left);
    }
}
```

## ⚙️ 六、复杂度分析

- **时间复杂度**：O(n) —— 每个节点最多访问一次
- **空间复杂度**：O(h) —— 递归栈深度，最坏 O(n)，平均 O(log n)

## 🧱 七、迭代（BFS/队列）版本（面试加分项）

```java
class Solution {
    public boolean isSymmetric(TreeNode root) {
        if (root == null) return true;

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root.left);
        queue.offer(root.right);

        while (!queue.isEmpty()) {
            TreeNode t1 = queue.poll();
            TreeNode t2 = queue.poll();

            if (t1 == null && t2 == null) continue;
            if (t1 == null || t2 == null) return false;
            if (t1.val != t2.val) return false;

            // 外侧与内侧成对入队
            queue.offer(t1.left);
            queue.offer(t2.right);
            queue.offer(t1.right);
            queue.offer(t2.left);
        }
        return true;
    }
}
```
