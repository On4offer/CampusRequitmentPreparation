### 一、题目解读

这道题的核心要求是：**在一个字符串中，找到所有连续的、没有重复字符的子串，并返回其中长度最大的那个子串的长度**。

需要特别注意两个关键点：

1. **子串 vs 子序列**：子串是字符串中**连续**的字符片段（如 "wke" 是 "pwwkew" 的子串），而子序列可以不连续（如 "pwke" 是子序列但不是子串），本题只关注子串。
2. **无重复字符**：子串中每个字符必须唯一，不能有重复（如 "bbbbb" 中最长无重复子串只能是单个 "b"）。

### 二、算法解释（滑动窗口法）

你提供的代码使用了「滑动窗口」（Sliding Window）算法，这是解决 “子串 / 子数组最值” 问题的经典高效方法，时间复杂度为 O(n)（n 是字符串长度），空间复杂度为 O(k)（k 是字符集大小，本题最多 128 个 ASCII 字符）。

#### 1. 核心思路

把字符串想象成一条线，用两个指针（`left` 和 `right`）框出一个 “窗口”，窗口内的字符就是当前考察的「无重复子串」：

- `right` 指针：主动向右移动，尝试扩大窗口，每次加入一个新字符。
- `left` 指针：当窗口内出现重复字符时，向右移动缩小窗口，直到窗口内无重复。
- 全程记录窗口的最大长度，就是最终答案。

#### 2. 代码逐行解释

```
class Solution {
    public int lengthOfLongestSubstring(String s) {
        // 1. 用HashSet存储当前窗口内的字符，快速判断是否重复
        Set<Character> charSet = new HashSet<>();
        // 2. 左指针，初始在窗口最左侧
        int left = 0;
        // 3. 记录最长无重复子串的长度，初始为0
        int maxLen = 0;

        // 4. 右指针遍历整个字符串，逐个字符加入窗口
        for(int right = 0; right < s.length(); right++){
            // 取出当前右指针指向的字符
            char currChar = s.charAt(right);

            // 5. 关键：如果当前字符已在窗口中，移动左指针缩小窗口，直到移除重复字符
            while(charSet.contains(currChar)){
                // 移除左指针指向的字符
                charSet.remove(s.charAt(left));
                // 左指针右移
                left++;
            }

            // 6. 将当前字符加入窗口（此时窗口内无重复）
            charSet.add(currChar);
            // 7. 更新最大长度：当前窗口长度（right - left + 1）和历史最大值取较大者
            maxLen = Math.max(maxLen, right - left + 1);
        }
        // 8. 返回最长长度
        return maxLen;
    }
}
```

#### 3. 示例推演（以 s = "pwwkew" 为例）

| right | currChar | 窗口状态（left~right） | charSet 内容 | maxLen |                             说明                             |
| :---: | :------: | :--------------------: | :----------: | :----: | :----------------------------------------------------------: |
|   0   |    p     |          0~0           |     {p}      |   1    |                      加入 p，窗口长度 1                      |
|   1   |    w     |          0~1           |    {p,w}     |   2    |                      加入 w，窗口长度 2                      |
|   2   |    w     |          2~2           |     {w}      |   2    | 发现 w 重复，移除 p、w（left 移到 2），重新加入 w，窗口长度 1 |
|   3   |    k     |          2~3           |    {w,k}     |   2    |                      加入 k，窗口长度 2                      |
|   4   |    e     |          2~4           |   {w,k,e}    |   3    |              加入 e，窗口长度 3（更新 maxLen）               |
|   5   |    w     |          3~5           |   {k,e,w}    |   3    |    发现 w 重复，移除 w（left 移到 3），加入 w，窗口长度 3    |

最终返回 maxLen = 3，符合示例答案。

### 总结

1. **核心算法**：滑动窗口（双指针），通过 `left` 和 `right` 动态调整窗口范围，保证窗口内无重复字符。
2. **关键优化**：用 HashSet 快速判断字符是否重复，避免暴力枚举所有子串（暴力法时间复杂度 O(n2)）。
3. **核心逻辑**：右指针扩窗，左指针去重，全程记录最大窗口长度。
