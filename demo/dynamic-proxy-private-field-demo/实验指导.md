# 实验指导：动态代理获取私有属性

本实验指导提供了多个可以手动操作的实验，帮助你深入理解动态代理获取私有属性的原理。

## 实验1：修改初始值观察变化

### 操作步骤
1. 打开 `UserServiceImpl.java`
2. 修改私有属性的初始值，例如：
   ```java
   private String privateField = "修改后的初始值";
   private Integer privateNum = 999;
   ```
3. 运行程序，观察输出

### 预期结果
- 代理对象读取到的初始值会发生变化
- 理解代理是在运行时读取目标对象的实际值

## 实验2：错误操作 - 操作proxy对象而不是target对象

### 操作步骤
1. 打开 `JdkProxyHandler.java`
2. 在 `invoke` 方法中，找到获取私有属性的代码
3. 将 `target.getClass().getDeclaredField(...)` 改为 `proxy.getClass().getDeclaredField(...)`
4. 将 `privateField.get(target)` 改为 `privateField.get(proxy)`
5. 运行程序，观察会发生什么

### 预期结果
- 可能会抛出 `NoSuchFieldException` 或 `IllegalAccessException`
- 理解为什么必须操作 target 对象而不是 proxy 对象

### 原因分析
- proxy 对象是动态生成的代理类实例，不包含目标类的私有属性
- target 对象才是真正包含私有属性的对象

3. 运行程序，观察会发生什么

### 预期结果

- 理解 `setAccessible(true)` 的作用：突破访问权限限制

## 实验4：修改final私有属性

### 操作步骤
1. 打开 `UserServiceImpl.java`
2. 添加一个 final 私有属性：
   ```java
   private final String finalField = "final值";
   ```
3. 在 `JdkProxyHandler.java` 中尝试修改这个字段：
   ```java
   Field finalField = target.getClass().getDeclaredField("finalField");
   finalField.setAccessible(true);
   finalField.set(target, "修改后的final值");
   ```
4. 运行程序，观察会发生什么

### 预期结果
- 对于基本类型和 String：可能抛出异常或修改失败
- 对于对象类型：可以修改对象内部的属性，但不能修改引用本身
- 理解 final 字段的限制

## 实验5：对比 getField 和 getDeclaredField

### 操作步骤
1. 在 `JdkProxyHandler.java` 中，尝试使用 `getField()` 而不是 `getDeclaredField()`：
   ```java
   // 错误示例
   Field field = target.getClass().getField("privateField");
   ```
2. 运行程序，观察会发生什么

### 预期结果
- 会抛出 `NoSuchFieldException`
- 理解 `getField()` 只能获取 public 字段，而 `getDeclaredField()` 可以获取所有字段（包括私有）

## 实验7：性能对比 - 缓存Field对象

### 操作步骤
1. 在 `JdkProxyHandler` 中添加静态缓存：
   ```java
   private static final Map<String, Field> fieldCache = new ConcurrentHashMap<>();
   ```
2. 修改获取字段的代码，先查缓存：
   ```java
   Field field = fieldCache.computeIfAbsent("privateField", name -> {
       try {
           Field f = target.getClass().getDeclaredField(name);
           f.setAccessible(true);
           return f;
       } catch (NoSuchFieldException e) {
           throw new RuntimeException(e);
       }
   });
   ```
3. 多次调用代理方法，观察性能差异（可以添加时间统计）

### 预期结果
- 缓存后性能会提升
- 理解反射操作的性能开销

## 实验8：获取所有私有属性

### 操作步骤
1. 创建一个方法，获取目标对象的所有私有属性：
   ```java
   public static void printAllPrivateFields(Object target) {
       Class<?> clazz = target.getClass();
       while (clazz != null) {
           Field[] fields = clazz.getDeclaredFields();
           for (Field field : fields) {
               if (Modifier.isPrivate(field.getModifiers())) {
                   field.setAccessible(true);
                   try {
                       System.out.println("私有属性：" + field.getName() + " = " + field.get(target));
                   } catch (IllegalAccessException e) {
                       e.printStackTrace();
                   }
               }
           }
           clazz = clazz.getSuperclass();
       }
   }
   ```
2. 在代理方法中调用这个方法

### 预期结果
- 可以看到目标对象的所有私有属性及其值
- 理解如何遍历所有私有属性
